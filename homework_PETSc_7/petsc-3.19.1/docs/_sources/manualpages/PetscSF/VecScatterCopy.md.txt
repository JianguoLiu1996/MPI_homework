# VecScatterCopy
Makes a copy of a scatter context. 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode VecScatterCopy(VecScatter sf, VecScatter *newsf)
```
Collective


## Input Parameter

- ***sf -*** the scatter context



## Output Parameter

- ***newsf -*** the context copy
