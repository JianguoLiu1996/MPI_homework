# PetscSFView
view a star forest 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode PetscSFView(PetscSF sf, PetscViewer viewer)
```
Collective


## Input Parameters

- ***sf -*** star forest
- ***viewer -*** viewer to display graph, for example `PETSC_VIEWER_STDOUT_WORLD`





## See Also
 `PetscSF`, `PetscViewer`, `PetscSFCreate()`, `PetscSFSetGraph()`
