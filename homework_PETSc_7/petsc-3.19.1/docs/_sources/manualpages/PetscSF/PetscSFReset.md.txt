# PetscSFReset
Reset a star forest so that different sizes or neighbors can be used 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode PetscSFReset(PetscSF sf)
```
Collective


## Input Parameter

- ***sf -*** star forest
