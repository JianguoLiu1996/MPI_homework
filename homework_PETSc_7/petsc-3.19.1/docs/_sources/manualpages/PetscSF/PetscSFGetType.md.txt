# PetscSFGetType
Get the `PetscSF` communication implementation 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode PetscSFGetType(PetscSF sf, PetscSFType *type)
```
Not Collective


## Input Parameter

- ***sf  -*** the `PetscSF` context



## Output Parameter

- ***type -*** the `PetscSF` type name
