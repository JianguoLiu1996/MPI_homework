# VecScatterSetUp
Sets up the `VecScatter` to be able to actually scatter information between vectors 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode VecScatterSetUp(VecScatter sf)
```
Collective


## Input Parameter

- ***sf -*** the scatter context
