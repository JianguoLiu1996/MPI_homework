# VecScatterGetType
Gets the vector scatter type name (as a string) from the `VecScatter`. 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode VecScatterGetType(VecScatter sf, VecScatterType *type)
```
Not Collective


## Input Parameter

- ***sf  -*** The vector scatter



## Output Parameter

- ***type -*** The vector scatter type name
