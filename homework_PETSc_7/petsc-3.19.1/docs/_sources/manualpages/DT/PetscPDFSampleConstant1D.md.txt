# PetscPDFSampleConstant1D
Sample uniformly from a uniform distribution on [-1, 1] in 1D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscPDFSampleConstant1D(const PetscReal p[], const PetscReal dummy[], PetscReal x[])
```
Not Collective


## Input Parameter

- ***p -*** A uniform variable on [0, 1]



## Output Parameter

- ***x -*** Coordinate in [-1, 1]
