# PetscDSGetFieldSize
Returns the size of the given field in the full space basis 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSGetFieldSize(PetscDS prob, PetscInt f, PetscInt *size)
```
Not Collective


## Input Parameters

- ***prob -*** The `PetscDS` object
- ***f -*** The field number



## Output Parameter

- ***size -*** The size
