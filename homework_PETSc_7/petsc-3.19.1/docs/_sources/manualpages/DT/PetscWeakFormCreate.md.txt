# PetscWeakFormCreate
Creates an empty `PetscWeakForm` object. 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscWeakFormCreate(MPI_Comm comm, PetscWeakForm *wf)
```
Collective


## Input Parameter

- ***comm -*** The communicator for the `PetscWeakForm` object



## Output Parameter

- ***wf -*** The `PetscWeakForm` object
