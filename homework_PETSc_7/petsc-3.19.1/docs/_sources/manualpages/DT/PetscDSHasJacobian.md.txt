# PetscDSHasJacobian
Checks that the Jacobian functions have been set 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSHasJacobian(PetscDS ds, PetscBool *hasJac)
```
Not Collective


## Input Parameter

- ***prob -*** The `PetscDS`



## Output Parameter

- ***hasJac -*** flag that pointwise function for the Jacobian has been set
