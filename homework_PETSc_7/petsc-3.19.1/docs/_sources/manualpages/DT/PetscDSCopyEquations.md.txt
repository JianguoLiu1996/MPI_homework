# PetscDSCopyEquations
Copy all pointwise function pointers to another `PetscDS` 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSCopyEquations(PetscDS prob, PetscDS newprob)
```
Not Collective


## Input Parameter

- ***prob -*** The `PetscDS` object



## Output Parameter

- ***newprob -*** The `PetscDS` copy
