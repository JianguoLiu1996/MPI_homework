# PetscPDFConstant2D
PDF for the uniform distribution in 2D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscPDFConstant2D(const PetscReal x[], const PetscReal dummy[], PetscReal p[])
```
Not Collective


## Input Parameter

- ***x -*** Coordinate in [-1, 1] x [-1, 1]



## Output Parameter

- ***p -*** The probability density at x
