# PetscPDFGaussian1D
PDF for the Gaussian distribution in 1D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscPDFGaussian1D(const PetscReal x[], const PetscReal scale[], PetscReal p[])
```
Not Collective


## Input Parameter

- ***x -*** Coordinate in [-\infty, \infty]



## Output Parameter

- ***p -*** The probability density at x
