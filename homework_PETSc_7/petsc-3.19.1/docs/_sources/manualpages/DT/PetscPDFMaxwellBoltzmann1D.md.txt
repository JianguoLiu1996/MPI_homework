# PetscPDFMaxwellBoltzmann1D
PDF for the Maxwell-Boltzmann distribution in 1D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscPDFMaxwellBoltzmann1D(const PetscReal x[], const PetscReal dummy[], PetscReal p[])
```
Not Collective


## Input Parameter

- ***x -*** Speed in [0, \infty]



## Output Parameter

- ***p -*** The probability density at x
