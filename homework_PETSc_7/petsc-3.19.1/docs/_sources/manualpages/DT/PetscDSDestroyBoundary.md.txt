# PetscDSDestroyBoundary
Remove all `DMBoundary` objects from the `PetscDS` 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSDestroyBoundary(PetscDS ds)
```
Not Collective


## Input Parameter

- ***ds -*** The `PetscDS` object
