# PetscWeakFormSetNumFields
Sets the number of fields 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscWeakFormSetNumFields(PetscWeakForm wf, PetscInt Nf)
```
Not Collective


## Input Parameters

- ***wf -*** The `PetscWeakForm` object
- ***Nf -*** The number of fields
