# PetscCDFConstant3D
CDF for the uniform distribution in 3D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscCDFConstant3D(const PetscReal x[], const PetscReal dummy[], PetscReal p[])
```
Not Collective


## Input Parameter

- ***x -*** Coordinate in [-1, 1] x [-1, 1] x [-1, 1]



## Output Parameter

- ***p -*** The cumulative probability at x
