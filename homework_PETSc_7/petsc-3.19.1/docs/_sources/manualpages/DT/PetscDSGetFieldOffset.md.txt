# PetscDSGetFieldOffset
Returns the offset of the given field in the full space basis 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSGetFieldOffset(PetscDS prob, PetscInt f, PetscInt *off)
```
Not Collective


## Input Parameters

- ***prob -*** The `PetscDS` object
- ***f -*** The field number



## Output Parameter

- ***off -*** The offset
