# PetscCDFConstant1D
CDF for the uniform distribution in 1D 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscCDFConstant1D(const PetscReal x[], const PetscReal dummy[], PetscReal p[])
```
Not Collective


## Input Parameter

- ***x -*** Coordinate in [-1, 1]



## Output Parameter

- ***p -*** The cumulative probability at x
