# PetscDTBinomial
Approximate the binomial coefficient "n choose k" 
## Input Parameters

- ***n -*** a non-negative integer
- ***k -*** an integer between 0 and n, inclusive



## Output Parameter

- ***binomial -*** approximation of the binomial coefficient n choose k
