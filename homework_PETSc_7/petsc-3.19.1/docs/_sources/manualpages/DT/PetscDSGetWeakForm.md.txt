# PetscDSGetWeakForm
Returns the weak form object 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSGetWeakForm(PetscDS ds, PetscWeakForm *wf)
```
Not Collective


## Input Parameter

- ***ds -*** The `PetscDS` object



## Output Parameter

- ***wf -*** The weak form object
