# PetscFormKeySort
Sorts an array of `PetscFormKey` in place in increasing order. 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscFormKeySort(PetscInt n, PetscFormKey arr[])
```
Not Collective


## Input Parameters

- ***n -*** number of values
- ***X -*** array of `PetscFormKey`





## See Also
 `PetscFormKey`, `PetscIntSortSemiOrdered()`, `PetscSortInt()`
