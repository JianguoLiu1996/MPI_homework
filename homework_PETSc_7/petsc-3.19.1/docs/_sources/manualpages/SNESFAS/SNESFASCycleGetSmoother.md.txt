# SNESFASCycleGetSmoother
Gets the smoother on a particular cycle level. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESFASCycleGetSmoother(SNES snes, SNES *smooth)
```
Logically Collective


## Input Parameter

- ***snes   -*** the `SNESFAS` nonlinear multigrid context



## Output Parameter

- ***smooth -*** the smoother
