# SNESFASSetLog
Sets or unsets time logging for various `SNESFAS` stages on all levels 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESFASSetLog(SNES snes, PetscBool flg)
```
Logically Collective


## Input Parameters

- ***snes   -*** the `SNESFAS` context
- ***flg    -*** monitor or not





## See Also
 `SNESFAS`, `SNESFASSetMonitor()`

advanced
