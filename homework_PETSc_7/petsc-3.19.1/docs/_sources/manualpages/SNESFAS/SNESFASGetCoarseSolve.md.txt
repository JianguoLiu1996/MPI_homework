# SNESFASGetCoarseSolve
Gets the coarsest solver. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESFASGetCoarseSolve(SNES snes, SNES *coarse)
```

## Input Parameter

- ***snes -*** the `SNESFAS` nonlinear multigrid context



## Output Parameter

- ***coarse -*** the coarse-level solver
