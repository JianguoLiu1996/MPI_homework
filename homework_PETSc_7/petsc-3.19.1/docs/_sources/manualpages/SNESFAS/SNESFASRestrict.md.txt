# SNESFASRestrict
restrict a `Vec` to the next coarser level 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESFASRestrict(SNES fine, Vec Xfine, Vec Xcoarse)
```
Collective


## Input Parameters

- ***fine -*** `SNES` from which to restrict
- ***Xfine -*** vector to restrict



## Output Parameter

- ***Xcoarse -*** result of restriction
