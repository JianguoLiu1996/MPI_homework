# DMSwarmRemovePointAtIndex
Removes a specific point from the `DMSWARM` 
## Synopsis
```
#include "petscdmswarm.h"   
PetscErrorCode DMSwarmRemovePointAtIndex(DM dm, PetscInt idx)
```
Not Collective


## Input parameters

- ***dm -*** a `DMSWARM`
- ***idx -*** index of point to remove
