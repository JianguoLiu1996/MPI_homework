# PetscViewerMathematicaPutVector
Send a vector to Mathematica via a `PETSCVIEWERMATHEMATICA` `PetscViewer` 
## Synopsis
```
PetscErrorCode PetscViewerMathematicaPutVector(PetscViewer viewer, Vec v)
```

## Input Parameters

- ***viewer -*** The Mathematica viewer
- ***v      -*** The vector





## See Also
 `PETSCVIEWERMATHEMATICA`, `VecView()`, `PetscViewerMathematicaGetVector()`

intermediate
