# PetscViewerVUFlushDeferred
Flushes the deferred write cache to the file. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscViewerVUFlushDeferred(PetscViewer viewer)
```
Not Collective


## Input Parameter

- ***viewer -*** The `PETSCVIEWERVU` `PetscViewer`
