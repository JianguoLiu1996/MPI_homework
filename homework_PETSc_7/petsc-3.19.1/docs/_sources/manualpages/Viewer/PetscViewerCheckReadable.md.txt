# PetscViewerCheckReadable
Check whether the viewer can be read from, generates an error if not 
## Synopsis
```
#include "petscviewer.h" 
PetscErrorCode PetscViewerCheckReadable(PetscViewer viewer)
```
Collective


## Input Parameter
