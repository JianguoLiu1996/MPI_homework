# PetscViewerView
Visualizes a viewer object. 
## Synopsis
```
#include "petscviewer.h" 
PetscErrorCode PetscViewerView(PetscViewer v, PetscViewer viewer)
```
Collective


## Input Parameters

- ***v -*** the viewer to be viewed
- ***viewer -*** visualization context
