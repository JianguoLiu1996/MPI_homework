# PetscViewerMathematicaClearName
Use the default name for objects communicated to Mathematica 
## Synopsis
```
PetscErrorCode PetscViewerMathematicaClearName(PetscViewer viewer)
```

## Input Parameter

- ***viewer -*** The Mathematica viewer
