# PetscSysFinalizePackage
This function destroys everything in the PETSc created internally in the system library portion of PETSc. It is called from `PetscFinalize()`. 
## Synopsis
```
PetscErrorCode PetscSysFinalizePackage(void)
```



## See Also
 `PetscSysInitializePackage()`, `PetscFinalize()`
