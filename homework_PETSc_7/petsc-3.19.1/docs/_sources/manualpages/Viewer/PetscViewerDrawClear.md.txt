# PetscViewerDrawClear
Clears a `PetscDraw` graphic associated with a `PetscViewer`. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscviewer.h" 
PetscErrorCode PetscViewerDrawClear(PetscViewer viewer)
```
Not Collective


## Input Parameter

- ***viewer -*** the `PetscViewer`
