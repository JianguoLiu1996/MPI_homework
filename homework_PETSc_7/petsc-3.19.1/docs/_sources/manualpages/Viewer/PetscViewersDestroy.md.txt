# PetscViewersDestroy
Destroys a set of `PetscViewer`s created with `PetscViewersCreate()`. 
## Synopsis
```
PetscErrorCode PetscViewersDestroy(PetscViewers *v)
```
Collective


## Input Parameter

- ***v -*** the `PetscViewer`s to be destroyed.
