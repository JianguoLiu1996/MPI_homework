# PetscSectionSymView
Displays a section symmetry 
## Synopsis
```
#include "petscsection.h"   
PetscErrorCode PetscSectionSymView(PetscSectionSym sym, PetscViewer viewer)
```
Collective


## Input Parameters

- ***sym -*** the index set
- ***viewer -*** viewer used to display the set, for example `PETSC_VIEWER_STDOUT_SELF`.





## See Also
