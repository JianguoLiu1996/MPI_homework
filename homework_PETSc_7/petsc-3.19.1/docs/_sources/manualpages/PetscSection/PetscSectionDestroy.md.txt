# PetscSectionDestroy
Frees a `PetscSection` 
## Synopsis
```
#include "petscsection.h"   
PetscErrorCode PetscSectionDestroy(PetscSection *s)
```
Not Collective


## Input Parameter

- ***s -*** the `PetscSection`
