# PetscSectionSymType
String with the name of a `PetscSectionSym` type. 
## Synopsis
```
typedef const char *PetscSectionSymType;

```



## Note
`PetscSectionSym` has no default implementation, but is used by `DM` in `PetscSectionSymCreateLabel()`.
