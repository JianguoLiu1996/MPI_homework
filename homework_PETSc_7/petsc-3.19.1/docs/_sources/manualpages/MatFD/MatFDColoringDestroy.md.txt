# MatFDColoringDestroy
Destroys a matrix coloring context that was created via `MatFDColoringCreate()`. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatFDColoringDestroy(MatFDColoring *c)
```
Collective


## Input Parameter

- ***c -*** coloring context
