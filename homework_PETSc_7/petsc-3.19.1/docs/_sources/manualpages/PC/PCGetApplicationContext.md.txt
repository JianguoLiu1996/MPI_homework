# PCGetApplicationContext
Gets the user-defined context for the linear solver. 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCGetApplicationContext(PC pc, void *usrP)
```
Not Collective


## Input Parameter

- ***pc -*** PC context



## Output Parameter

- ***usrP -*** user context





## See Also
 `PCSetApplicationContext()`

intermediate
