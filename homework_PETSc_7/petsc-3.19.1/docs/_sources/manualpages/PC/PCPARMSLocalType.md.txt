# PCPARMSLocalType
Determines the local preconditioner method in `PCPARMS` 
## Synopsis
```
typedef enum {
  PC_PARMS_LOCAL_ILU0,
  PC_PARMS_LOCAL_ILUK,
  PC_PARMS_LOCAL_ILUT,
  PC_PARMS_LOCAL_ARMS
} PCPARMSLocalType;
```
