# PCSetApplicationContext
Sets the optional user-defined context for the linear solver. 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCSetApplicationContext(PC pc, void *usrP)
```
Logically Collective


## Input Parameters

- ***pc -*** the PC context
- ***usrP -*** optional user context
