# PCRedundantGetKSP
Gets the less parallel `KSP` created by the redundant `PC`. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCRedundantGetKSP(PC pc, KSP *innerksp)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***innerksp -*** the `KSP` on the smaller set of processes

 `PCREDUNDANT`
