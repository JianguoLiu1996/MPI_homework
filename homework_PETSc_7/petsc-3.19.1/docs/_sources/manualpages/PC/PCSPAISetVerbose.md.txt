# PCSPAISetVerbose
verbosity level for the `PCSPAI` preconditioner 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCSPAISetVerbose(PC pc, PetscInt verbose)
```

## Input Parameters

- ***pc -*** the preconditioner
- ***n -*** level (default 1)



## Note
print parameters, timings and matrix statistics
