# PCMGCycleType
Use V-cycle or W-cycle 
## Synopsis
```
typedef enum {
  PC_MG_CYCLE_V = 1,
  PC_MG_CYCLE_W = 2
} PCMGCycleType;
```

## Values

- ***`PC_MG_V_CYCLE` -*** use the v cycle
- ***`PC_MG_W_CYCLE` -*** use the w cycle
