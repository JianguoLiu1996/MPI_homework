# PCMGGetCoarseSolve
Gets the solver context to be used on the coarse grid. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCMGGetCoarseSolve(PC pc, KSP *ksp)
```
Not Collective


## Input Parameter

- ***pc -*** the multigrid context



## Output Parameter

- ***ksp -*** the coarse grid solver context
