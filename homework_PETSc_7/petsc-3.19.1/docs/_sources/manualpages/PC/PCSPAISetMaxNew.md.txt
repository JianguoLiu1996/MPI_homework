# PCSPAISetMaxNew
set maximum number of new nonzero candidates per step in `PCSPAI` preconditioner 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCSPAISetMaxNew(PC pc, PetscInt maxnew1)
```

## Input Parameters

- ***pc -*** the preconditioner
- ***n -*** maximum number (default 5)
