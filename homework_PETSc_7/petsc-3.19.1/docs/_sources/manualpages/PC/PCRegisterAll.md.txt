# PCRegisterAll
Registers all of the preconditioners in the PC package. 
## Synopsis
```
#include "petscpc.h"   
PetscErrorCode PCRegisterAll(void)
```
Not Collective


## Input Parameter

- ***path -*** the library where the routines are to be found (optional)
