# PCDestroy
Destroys `PC` context that was created with `PCCreate()`. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCDestroy(PC *pc)
```
Collective


## Input Parameter

- ***pc -*** the preconditioner context





## See Also
 `PC`, `PCCreate()`, `PCSetUp()`

## Level
developer
