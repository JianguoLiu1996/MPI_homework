# PCGAMGGetType
Get the type of algorithm `PCGAMG` will use 
## Synopsis
```
#include "petscpc.h" 
#include "petscksp.h" 
PetscErrorCode PCGAMGGetType(PC pc, PCGAMGType *type)
```
Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***type -*** the type of algorithm used
