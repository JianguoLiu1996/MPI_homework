# PCDeflationGetPC
Returns the additional preconditioner M^{-1}. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCDeflationGetPC(PC pc, PC *apc)
```
Not Collective


## Input Parameter

- ***pc  -*** the preconditioner context



## Output Parameter

- ***apc -*** additional preconditioner
