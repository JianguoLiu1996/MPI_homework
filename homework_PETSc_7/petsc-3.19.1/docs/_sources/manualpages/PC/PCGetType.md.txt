# PCGetType
Gets the PC method type and name (as a string) from the PC context. 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCGetType(PC pc, PCType *type)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***type -*** name of preconditioner method





## See Also
 `PCSetType()`

intermediate
