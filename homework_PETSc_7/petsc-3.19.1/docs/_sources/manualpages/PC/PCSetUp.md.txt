# PCSetUp
Prepares for the use of a preconditioner. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCSetUp(PC pc)
```
Collective


## Input Parameter

- ***pc -*** the preconditioner context





## See Also
 `PC`, `PCCreate()`, `PCApply()`, `PCDestroy()`
