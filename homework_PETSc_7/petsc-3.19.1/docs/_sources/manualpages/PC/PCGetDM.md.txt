# PCGetDM
Gets the DM that may be used by some preconditioners 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCGetDM(PC pc, DM *dm)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***dm -*** the dm





## See Also
 `PCSetDM()`, `KSPSetDM()`, `KSPGetDM()`

## Level
intermediate
