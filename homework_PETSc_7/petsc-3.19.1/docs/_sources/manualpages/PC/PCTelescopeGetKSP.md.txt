# PCTelescopeGetKSP
Gets the `KSP` created by the telescoping `PC`. 
## Synopsis
```
#include "petscksp.h" 
#include "petscdm.h" 
PetscErrorCode PCTelescopeGetKSP(PC pc, KSP *subksp)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***subksp -*** the `KSP` defined the smaller set of processes





## See Also
 `PCTELESCOPE`

advanced
