# PCFactorGetShiftAmount
Gets the tolerance used to define a zero privot 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCFactorGetShiftAmount(PC pc, PetscReal *shift)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***shift -*** how much to shift the diagonal entry
