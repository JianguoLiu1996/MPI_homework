# PCMGGetAdaptCoarseSpaceType
Get the type of adaptive coarse space. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCMGGetAdaptCoarseSpaceType(PC pc, PCMGCoarseSpaceType *ctype)
```
Not Collective


## Input Parameter

- ***pc    -*** the multigrid context



## Output Parameter

- ***ctype -*** the type of coarse space
