# PCFactorGetZeroPivot
Gets the tolerance used to define a zero privot 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCFactorGetZeroPivot(PC pc, PetscReal *pivot)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***pivot -*** the tolerance
