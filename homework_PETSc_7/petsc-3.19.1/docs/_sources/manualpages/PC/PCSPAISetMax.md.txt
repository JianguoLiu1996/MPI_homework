# PCSPAISetMax
set the size of various working buffers in the `PCSPAI` preconditioner 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCSPAISetMax(PC pc, PetscInt max1)
```

## Input Parameters

- ***pc -*** the preconditioner
- ***n -*** size (default is 5000)





## See Also
 `PCSPAI`, `PCSetType()`

## Level
intermediate
