# PCFieldSplitSchurRestoreS
returns the `MATSCHURCOMPLEMENT` matrix used by this `PC` 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCFieldSplitSchurRestoreS(PC pc, Mat *S)
```
Not Collective


## Input Parameters

- ***pc      -*** the preconditioner context
- ***S       -*** the Schur complement matrix
