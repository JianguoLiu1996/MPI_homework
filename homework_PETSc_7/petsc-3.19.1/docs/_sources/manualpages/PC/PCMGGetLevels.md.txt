# PCMGGetLevels
Gets the number of levels to use with `PCMG`. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCMGGetLevels(PC pc, PetscInt *levels)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output parameter

- ***levels -*** the number of levels





## See Also
 `PCMG`, `PCMGSetLevels()`

advanced
