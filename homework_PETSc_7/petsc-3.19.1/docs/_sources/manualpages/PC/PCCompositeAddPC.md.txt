# PCCompositeAddPC
Adds another `PC` to the composite `PC`. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCCompositeAddPC(PC pc, PC subpc)
```
Collective


## Input Parameters

- ***pc    -*** the preconditioner context
- ***subpc -*** the new preconditioner
