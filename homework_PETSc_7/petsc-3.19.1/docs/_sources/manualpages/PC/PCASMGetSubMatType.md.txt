# PCASMGetSubMatType
Gets the matrix type used for `PCASM` subsolves, as a string. 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCASMGetSubMatType(PC pc, MatType *sub_mat_type)
```
Not Collective


## Input Parameter

- ***pc -*** the `PC`



## Output Parameter

- ***pc_asm_sub_mat_type -*** name of matrix type
