# PCViewFromOptions
View from the `PC` based on options in the database 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCViewFromOptions(PC A, PetscObject obj, const char name[])
```
Collective


## Input Parameters

- ***A -*** the `PC` context
- ***obj -*** Optional object that provides the options prefix
- ***name -*** command line option
