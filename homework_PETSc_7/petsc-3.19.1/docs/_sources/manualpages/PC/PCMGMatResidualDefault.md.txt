# PCMGMatResidualDefault
Default routine to calculate the residual. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCMGMatResidualDefault(Mat mat, Mat b, Mat x, Mat r)
```
Collective


## Input Parameters

- ***mat -*** the matrix
- ***b   -*** the right-hand-side
- ***x   -*** the approximate solution



## Output Parameter

- ***r -*** location to store the residual
