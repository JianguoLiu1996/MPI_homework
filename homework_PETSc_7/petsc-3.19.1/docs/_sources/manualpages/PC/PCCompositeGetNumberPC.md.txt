# PCCompositeGetNumberPC
Gets the number of `PC` objects in the composite `PC`. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode PCCompositeGetNumberPC(PC pc, PetscInt *num)
```
Not Collective


## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***num -*** the number of sub pcs
