# PCGAMGCreateGraph
Creates a graph that is used by the ``PCGAMGType`` in the coarsening process 
## Synopsis
```
#include "petscpc.h" 
#include "petscksp.h" 
PetscErrorCode PCGAMGCreateGraph(PC pc, Mat A, Mat *G)
```

## Input Parameters

- ***pc -*** the `PCGAMG`
- ***A -*** the matrix, for any level



## Output Parameter

- ***G -*** the graph





## See Also
 `PCGAMGType`, `PCGAMG`, `PCGAMGSetType()`

advanced
