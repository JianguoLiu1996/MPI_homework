# PCHYPREGetType
Gets which hypre preconditioner you are using 
## Synopsis
```
#include "petscpc.h" 
PetscErrorCode PCHYPREGetType(PC pc, const char *name[])
```

## Input Parameter

- ***pc -*** the preconditioner context



## Output Parameter

- ***name -*** either  euclid, pilut, parasails, boomeramg, ams, ads
