# PFType
Type of PETSc mathematical function, a string name 
## Synopsis
```
typedef const char *PFType;
#define PFCONSTANT "constant"
#define PFMAT      "mat"
#define PFSTRING   "string"
#define PFQUICK    "quick"
#define PFIDENTITY "identity"
#define PFMATLAB   "matlab"

```
