# PFDestroy
Destroys `PF` context that was created with `PFCreate()`. 
## Synopsis
```
#include "petscpf.h" 
PetscErrorCode PFDestroy(PF *pf)
```
Collective


## Input Parameter

- ***pf -*** the function context





## See Also
 `PF`, `PFCreate()`, `PFSet()`, `PFSetType()`
