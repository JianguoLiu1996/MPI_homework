# DMCompositeAddDM
adds a `DM` vector to a `DMCOMPOSITE` 
## Synopsis
```
#include "petscdmcomposite.h"  
PetscErrorCode DMCompositeAddDM(DM dmc, DM dm)
```
Collective


## Input Parameters

- ***dmc -*** the  `DMCOMPOSITE` object
- ***dm -*** the `DM` object
