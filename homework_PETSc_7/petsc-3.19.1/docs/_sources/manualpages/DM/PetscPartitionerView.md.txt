# PetscPartitionerView
Views a `PetscPartitioner` 
## Synopsis
```
#include "petscpartitioner.h" 
PetscErrorCode PetscPartitionerView(PetscPartitioner part, PetscViewer v)
```
Collective


## Input Parameters

- ***part -*** the `PetscPartitioner` object to view
- ***v    -*** the viewer
