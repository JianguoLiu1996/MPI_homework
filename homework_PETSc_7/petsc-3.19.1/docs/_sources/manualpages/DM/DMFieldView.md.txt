# DMFieldView
view a `DMField` 
## Synopsis
```
#include "petscdmfield.h" 
#include "petscdmfield.h" 
PetscErrorCode DMFieldView(DMField field, PetscViewer viewer)
```
Collective


## Input Parameters

- ***field -*** `DMField`
- ***viewer -*** viewer to display field, for example `PETSC_VIEWER_STDOUT_WORLD`
