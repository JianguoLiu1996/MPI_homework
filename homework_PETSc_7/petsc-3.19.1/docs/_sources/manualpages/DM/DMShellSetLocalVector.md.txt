# DMShellSetLocalVector
sets a template local vector associated with the `DMSHELL` 
## Synopsis
```
#include "petscdmshell.h"  
PetscErrorCode DMShellSetLocalVector(DM dm, Vec X)
```
Logically Collective


## Input Parameters

- ***dm -*** `DMSHELL`
- ***X -*** template vector
