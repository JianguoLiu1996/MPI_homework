# DMAddLabel
Add the label to this `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMAddLabel(DM dm, DMLabel label)
```
Not Collective


## Input Parameters

- ***dm   -*** The `DM` object
- ***label -*** The `DMLabel`
