# PetscFERegisterAll
Registers all of the PetscFE components in the PetscFE package. 
## Synopsis
```
#include "petscdm.h"  
#include "petscdmplex.h"  
#include "petscfe.h"  
PetscErrorCode PetscFERegisterAll(void)
```
Not Collective


## Input parameter

- ***path -*** The dynamic library path

 `PetscFERegister()`, `PetscFERegisterDestroy()`
