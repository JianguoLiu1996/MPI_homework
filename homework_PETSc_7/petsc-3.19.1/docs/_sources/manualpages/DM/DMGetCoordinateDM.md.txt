# DMGetCoordinateDM
Gets the `DM` that prescribes coordinate layout and scatters between global and local coordinates 
## Synopsis
```
#include "petscdm.h"          
PetscErrorCode DMGetCoordinateDM(DM dm, DM *cdm)
```
Collective


## Input Parameter

- ***dm -*** the `DM`



## Output Parameter

- ***cdm -*** coordinate `DM`
