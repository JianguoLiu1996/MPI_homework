# DMSetNumFields
Set the number of fields in the `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetNumFields(DM dm, PetscInt numFields)
```
Logically Collective


## Input Parameters

- ***dm -*** The `DM`
- ***Nf -*** The number of fields
