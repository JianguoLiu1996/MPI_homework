# DMRestoreGlobalVector
Returns a PETSc vector that obtained from `DMGetGlobalVector()`. Do not use with vector obtained via `DMCreateGlobalVector()`. 
## Synopsis
```
#include "petscdm.h" 
PetscErrorCode DMRestoreGlobalVector(DM dm, Vec *g)
```
Not Collective


## Input Parameters

- ***dm -*** the `DM`
- ***g -*** the global vector
