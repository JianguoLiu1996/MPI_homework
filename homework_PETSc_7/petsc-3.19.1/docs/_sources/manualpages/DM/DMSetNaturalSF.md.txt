# DMSetNaturalSF
Set the PetscSF encoding the map back to the original mesh ordering 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetNaturalSF(DM dm, PetscSF sf)
```

## Input Parameters

- ***dm -*** The DM
- ***sf -*** The PetscSF
