# DMGetLabelIdIS
Get the `DMLabelGetValueIS()` from a `DMLabel` in the `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetLabelIdIS(DM dm, const char name[], IS *ids)
```
Not Collective


## Input Parameters

- ***mesh -*** The `DM` object
- ***name -*** The label name



## Output Parameter

- ***ids -*** The integer ids, or `NULL` if the label does not exist
