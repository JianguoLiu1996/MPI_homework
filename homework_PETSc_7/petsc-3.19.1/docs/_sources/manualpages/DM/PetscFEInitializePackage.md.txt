# PetscFEInitializePackage
This function initializes everything in the FE package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to PetscSpaceCreate() when using static libraries. 
## Synopsis
```
PetscErrorCode PetscFEInitializePackage(void)
```
