# DMDirection
Indicates a coordinate direction 
## Synopsis
```
typedef enum {
  DM_X,
  DM_Y,
  DM_Z
} DMDirection;
```

## Values

- ***`DM_X` -*** the x coordinate direction
- ***`DM_Y` -*** the y coordinate direction
- ***`DM_Z` -*** the z coordinate direction
