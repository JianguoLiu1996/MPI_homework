# DMInitializePackage
This function initializes everything in the DM package. It is called from PetscDLLibraryRegister_petscdm() when using dynamic libraries, and on the first call to AOCreate() or DMDACreate() when using shared or static libraries. 
## Synopsis
```
PetscErrorCode DMInitializePackage(void)
```
