# DMHasLabel
Determine whether the `DM` has a label of a given name 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMHasLabel(DM dm, const char name[], PetscBool *hasLabel)
```
Not Collective


## Input Parameters

- ***dm   -*** The `DM` object
- ***name -*** The label name



## Output Parameter

- ***hasLabel -*** `PETSC_TRUE` if the label is present
