# DMRefineHierarchy
Refines a `DM` object, all levels at once 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMRefineHierarchy(DM dm, PetscInt nlevels, DM dmf[])
```
Collective


## Input Parameters

- ***dm -*** the `DM` object
- ***nlevels -*** the number of levels of refinement



## Output Parameter

- ***dmf -*** the refined `DM` hierarchy
