# DMGetCoarseDM
Get the coarse `DM`from which this `DM` was obtained by refinement 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetCoarseDM(DM dm, DM *cdm)
```
Not Collective


## Input Parameter

- ***dm -*** The `DM` object



## Output Parameter

- ***cdm -*** The coarse `DM`
