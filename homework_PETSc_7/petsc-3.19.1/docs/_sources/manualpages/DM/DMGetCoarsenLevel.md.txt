# DMGetCoarsenLevel
Gets the number of coarsenings that have generated this `DM`. 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetCoarsenLevel(DM dm, PetscInt *level)
```
Not Collective


## Input Parameter

- ***dm -*** the `DM` object



## Output Parameter

- ***level -*** number of coarsenings
