# DMCopyFields
Copy the discretizations for the `DM` into another `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMCopyFields(DM dm, DM newdm)
```
Collective


## Input Parameter

- ***dm -*** The `DM`



## Output Parameter

- ***newdm -*** The `DM`
