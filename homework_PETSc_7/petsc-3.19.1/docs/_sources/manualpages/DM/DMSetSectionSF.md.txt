# DMSetSectionSF
Set the `PetscSF` encoding the parallel dof overlap for the `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetSectionSF(DM dm, PetscSF sf)
```

## Input Parameters

- ***dm -*** The `DM`
- ***sf -*** The `PetscSF`





## Note
Any previous `PetscSF` is destroyed
