# DMGetLabelByNum
Return the nth label on a `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetLabelByNum(DM dm, PetscInt n, DMLabel *label)
```
Not Collective


## Input Parameters

- ***dm -*** The `DM` object
- ***n  -*** the label number



## Output Parameter

- ***label -*** the label
