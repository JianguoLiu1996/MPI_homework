# PetscDSInitializePackage
This function initializes everything in the DS package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to PetscDSCreate() when using static libraries. 
## Synopsis
```
PetscErrorCode PetscDSInitializePackage(void)
```
