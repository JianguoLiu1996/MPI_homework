# DMSetCoordinateDim
Set the dimension of the embedding space for coordinate values. 
## Synopsis
```
#include "petscdm.h"          
PetscErrorCode DMSetCoordinateDim(DM dm, PetscInt dim)
```
Not Collective


## Input Parameters

- ***dm  -*** The `DM` object
- ***dim -*** The embedding dimension
