# DMGetDS
Get the default `PetscDS` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetDS(DM dm, PetscDS *ds)
```
Not Collective


## Input Parameter

- ***dm -*** The `DM`



## Output Parameter

- ***ds -*** The default `PetscDS`
