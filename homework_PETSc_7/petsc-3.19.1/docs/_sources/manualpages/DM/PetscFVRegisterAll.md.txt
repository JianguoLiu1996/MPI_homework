# PetscFVRegisterAll
Registers all of the PetscFV components in the PetscFV package. 
## Synopsis
```
#include "petscdm.h"  
#include "petscdmplex.h"  
#include "petscfe.h"  
#include "petscfv.h"  
PetscErrorCode PetscFVRegisterAll(void)
```
Not Collective


## Input parameter

- ***path -*** The dynamic library path





## See Also
 `PetscFVRegister()`, `PetscFVRegisterDestroy()`

advanced
