# DMSetFineDM
Set the fine mesh from which this was obtained by coarsening 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetFineDM(DM dm, DM fdm)
```

## Input Parameters

- ***dm -*** The `DM` object
- ***fdm -*** The fine `DM`





## Note
Normally this is set automatically by `DMCoarsen()`
