# DMClearLabelStratum
Remove all points from a stratum from a `DMLabel` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMClearLabelStratum(DM dm, const char name[], PetscInt value)
```
Not Collective


## Input Parameters

- ***dm   -*** The `DM` object
- ***name -*** The label name
- ***value -*** The label value for this point
