# DMGetLabelOutput
Get the output flag for a given label 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMGetLabelOutput(DM dm, const char name[], PetscBool *output)
```
Not Collective


## Input Parameters

- ***dm   -*** The `DM` object
- ***name -*** The label name



## Output Parameter

- ***output -*** The flag for output
