# DMRegisterAll
Registers all of the DM components in the DM package. 
## Synopsis
```
#include "petscdm.h"  
#include "petscdmplex.h"  
PetscErrorCode DMRegisterAll(void)
```
Not Collective


## Input parameter

- ***path -*** The dynamic library path





## See Also
 `DMRegister()`, `DMRegisterDestroy()`

advanced
