# DMShellSetContext
set some data to be usable by this `DMSHELL` 
## Synopsis
```
#include "petscdmshell.h"  
PetscErrorCode DMShellSetContext(DM dm, void *ctx)
```
Collective


## Input Parameters

- ***dm -*** `DMSHELL`
- ***ctx -*** the context
