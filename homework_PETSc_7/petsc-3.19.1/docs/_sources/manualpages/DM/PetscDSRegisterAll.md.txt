# PetscDSRegisterAll
Registers all of the PetscDS components in the PetscDS package. 
## Synopsis
```
#include "petscdm.h"  
#include "petscdmplex.h"  
#include "petscfe.h"  
#include "petscfv.h"  
#include "petscds.h"  
PetscErrorCode PetscDSRegisterAll(void)
```
Not Collective


## Input parameter

- ***path -*** The dynamic library path





## See Also
 `PetscDSRegister()`, `PetscDSRegisterDestroy()`

advanced
