# DMSetCoordinateDM
Sets the `DM` that prescribes coordinate layout and scatters between global and local coordinates 
## Synopsis
```
#include "petscdm.h"          
PetscErrorCode DMSetCoordinateDM(DM dm, DM cdm)
```
Logically Collective


## Input Parameters

- ***dm -*** the `DM`
- ***cdm -*** coordinate `DM`
