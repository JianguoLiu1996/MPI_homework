# AOGetType
Gets the `AO` type name (as a string) from the AO. 
## Synopsis
```
#include "petscao.h"  
PetscErrorCode AOGetType(AO ao, AOType *type)
```
Not Collective


## Input Parameter

- ***ao  -*** The vector



## Output Parameter

- ***type -*** The `AO` type name
