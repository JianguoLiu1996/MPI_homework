# AOViewFromOptions
View an `AO` based on values in the options database 
## Synopsis
```
#include "petscao.h" 
PetscErrorCode AOViewFromOptions(AO ao, PetscObject obj, const char name[])
```
Collective


## Input Parameters

- ***ao -*** the application ordering context
- ***obj -*** Optional object
- ***name -*** command line option
