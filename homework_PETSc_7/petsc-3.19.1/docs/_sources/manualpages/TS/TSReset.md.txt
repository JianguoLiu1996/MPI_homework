# TSReset
Resets a `TS` context and removes any allocated `Vec`s and `Mat`s. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSReset(TS ts)
```
Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
