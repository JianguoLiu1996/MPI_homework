# TSThetaSetTheta
Set the abscissa of the stage in (0,1]  for `TSTHETA` 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSThetaSetTheta(TS ts, PetscReal theta)
```
Not Collective


## Input Parameters

- ***ts -*** timestepping context
- ***theta -*** stage abscissa



## Options Database Key

- ***-ts_theta_theta <theta> -*** set theta
