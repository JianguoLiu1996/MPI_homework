# TSSSPGetNumStages
get the number of stages in the `TSSSP` time integration scheme 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSSSPGetNumStages(TS ts, PetscInt *nstages)
```
Logically Collective


## Input Parameter

- ***ts -*** time stepping object



## Output Parameter

- ***nstages -*** number of stages
