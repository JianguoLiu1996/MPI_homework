# TSARKIMEXRegisterDestroy
Frees the list of schemes that were registered by `TSARKIMEXRegister()`. 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSARKIMEXRegisterDestroy(void)
```
Not Collective
