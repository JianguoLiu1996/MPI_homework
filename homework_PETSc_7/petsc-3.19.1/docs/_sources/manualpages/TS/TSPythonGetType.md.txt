# TSPythonGetType
Get the type of a `TS` object implemented in Python. 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode TSPythonGetType(TS ts, const char *pyname[])
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context



## Output Parameter

- ***pyname -*** full dotted Python name [package].module[.{class|function}]
