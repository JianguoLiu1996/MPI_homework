# TSIRKRegisterAll
Registers all of the implicit Runge-Kutta methods in `TSIRK` 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSIRKRegisterAll(void)
```
Not Collective, but should be called by all processes which will need the schemes to be registered
