# TSSetTime
Allows one to reset the time. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSSetTime(TS ts, PetscReal t)
```
Logically Collective


## Input Parameters

- ***ts -*** the `TS` context obtained from `TSCreate()`
- ***time -*** the time
