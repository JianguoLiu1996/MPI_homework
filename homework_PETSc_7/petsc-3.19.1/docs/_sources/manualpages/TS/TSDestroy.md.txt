# TSDestroy
Destroys the timestepper context that was created with `TSCreate()`. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSDestroy(TS *ts)
```
Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
