# TSGetPrevTime
Gets the starting time of the previously completed step. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetPrevTime(TS ts, PetscReal *t)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`



## Output Parameter

- ***t  -*** the previous time
