# TSRKRegisterAll
Registers all of the Runge-Kutta explicit methods in `TSRK` 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSRKRegisterAll(void)
```
Not Collective, but should be called by all processes which will need the schemes to be registered
