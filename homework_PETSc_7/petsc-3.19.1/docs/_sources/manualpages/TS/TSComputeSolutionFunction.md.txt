# TSComputeSolutionFunction
Evaluates the solution function. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSComputeSolutionFunction(TS ts, PetscReal t, Vec U)
```
Collective


## Input Parameters

- ***ts -*** the `TS` context
- ***t -*** current time



## Output Parameter

- ***U -*** the solution
