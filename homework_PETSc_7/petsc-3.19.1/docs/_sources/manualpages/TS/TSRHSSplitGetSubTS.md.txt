# TSRHSSplitGetSubTS
Get the sub-`TS` by split name. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSRHSSplitGetSubTS(TS ts, const char splitname[], TS *subts)
```
Logically Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`



## Output Parameters

- ***splitname -*** the number of the split
- ***subts -*** the sub-`TS`
