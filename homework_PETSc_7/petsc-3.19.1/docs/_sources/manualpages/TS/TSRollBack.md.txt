# TSRollBack
Rolls back one time step 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSRollBack(TS ts)
```
Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
