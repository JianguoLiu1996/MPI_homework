# TSMonitorEnvelopeCtxDestroy
Destroys a context that was created  with `TSMonitorEnvelopeCtxCreate()`. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSMonitorEnvelopeCtxDestroy(TSMonitorEnvelopeCtx *ctx)
```
Collective


## Input Parameter

- ***ctx -*** the monitor context
