# DMTSCopy
copies the information in a `DMTS` to another `DMTS` 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode DMTSCopy(DMTS kdm, DMTS nkdm)
```
Not Collective


## Input Parameters

- ***kdm -*** Original `DMTS`
- ***nkdm -*** `DMTS` to receive the data, should have been created with `DMTSCreate()`
