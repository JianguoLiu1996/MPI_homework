# TSRosWSetType
Set the type of Rosenbrock-W, `TSROSW`, scheme 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSRosWSetType(TS ts, TSRosWType roswtype)
```
Logically Collective


## Input Parameters

- ***ts -*** timestepping context
- ***roswtype -*** type of Rosenbrock-W scheme
