# TSAdaptReset
Resets a `TSAdapt` context to its defaults 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode TSAdaptReset(TSAdapt adapt)
```
Collective


## Input Parameter

- ***adapt -*** the `TSAdapt` context obtained from `TSGetAdapt()` or `TSAdaptCreate()`
