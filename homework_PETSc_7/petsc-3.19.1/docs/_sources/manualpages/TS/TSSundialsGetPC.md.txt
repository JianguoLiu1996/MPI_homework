# TSSundialsGetPC
Extract the PC context from a time-step context for `TSSUNDIALS`. 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode TSSundialsGetPC(TS ts, PC *pc)
```

## Input Parameter

- ***ts -*** the time-step context



## Output Parameter

- ***pc -*** the preconditioner context
