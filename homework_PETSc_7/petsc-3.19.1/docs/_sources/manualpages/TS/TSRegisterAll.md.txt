# TSRegisterAll
Registers all of the timesteppers in the `TS` package. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSRegisterAll(void)
```
Not Collective


## Input parameter

- ***path -*** The dynamic library path
