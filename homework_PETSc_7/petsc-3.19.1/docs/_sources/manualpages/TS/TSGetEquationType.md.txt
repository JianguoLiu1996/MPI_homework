# TSGetEquationType
Gets the type of the equation that `TS` is solving. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetEquationType(TS ts, TSEquationType *equation_type)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context



## Output Parameter

- ***equation_type -*** see `TSEquationType`
