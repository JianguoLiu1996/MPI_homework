# TSGetMaxTime
Gets the maximum (or final) time for timestepping. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetMaxTime(TS ts, PetscReal *maxtime)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`



## Output Parameter

- ***maxtime -*** final time to step to
