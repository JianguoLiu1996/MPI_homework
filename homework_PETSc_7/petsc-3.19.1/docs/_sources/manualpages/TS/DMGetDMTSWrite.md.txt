# DMGetDMTSWrite
get write access to private `DMTS` context from a `DM` 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode DMGetDMTSWrite(DM dm, DMTS *tsdm)
```
Not Collective


## Input Parameter

- ***dm -*** `DM` to be used with `TS`



## Output Parameter

- ***tsdm -*** private `DMTS` context
