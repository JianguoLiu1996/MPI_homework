# TSSSPGetType
get the `TSSSP` time integration scheme 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSSSPGetType(TS ts, TSSSPType *type)
```
Logically Collective


## Input Parameter

- ***ts -*** time stepping object



## Output Parameter

- ***type -*** type of scheme being used
