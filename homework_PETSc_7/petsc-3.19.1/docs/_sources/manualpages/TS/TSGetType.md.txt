# TSGetType
Gets the `TS` method type (as a string). 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetType(TS ts, TSType *type)
```
Not Collective


## Input Parameter

- ***ts -*** The `TS`



## Output Parameter

- ***type -*** The name of `TS` method
