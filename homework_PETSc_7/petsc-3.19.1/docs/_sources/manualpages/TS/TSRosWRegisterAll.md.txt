# TSRosWRegisterAll
Registers all of the Rosenbrock-W methods in `TSROSW` 
## Synopsis
```
#include "petscts.h"   
PetscErrorCode TSRosWRegisterAll(void)
```
Not Collective, but should be called by all processes which will need the schemes to be registered
