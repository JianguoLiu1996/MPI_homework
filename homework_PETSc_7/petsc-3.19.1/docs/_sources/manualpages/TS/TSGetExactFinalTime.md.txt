# TSGetExactFinalTime
Gets the exact final time option set with `TSSetExactFinalTime()` 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetExactFinalTime(TS ts, TSExactFinalTimeOption *eftopt)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context



## Output Parameter

- ***eftopt -*** exact final time option
