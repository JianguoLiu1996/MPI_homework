# TSGetStepNumber
Gets the number of time steps completed. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetStepNumber(TS ts, PetscInt *steps)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`



## Output Parameter

- ***steps -*** number of steps completed so far
