# TSGetMaxSteps
Gets the maximum number of steps to use. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSGetMaxSteps(TS ts, PetscInt *maxsteps)
```
Not Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`



## Output Parameter

- ***maxsteps -*** maximum number of steps to use
