# VecMean
Computes the arithmetic mean of all the components of a vector. 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecMean(Vec v, PetscScalar *mean)
```
Collective


## Input Parameter

- ***v -*** the vector



## Output Parameter

- ***mean -*** the result
