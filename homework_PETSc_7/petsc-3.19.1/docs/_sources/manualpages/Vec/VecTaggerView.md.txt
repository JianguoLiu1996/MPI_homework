# VecTaggerView
view a `VecTagger` context 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecTaggerView(VecTagger tagger, PetscViewer viewer)
```
Collective


## Input Parameters

- ***tagger -*** vec tagger
- ***viewer -*** viewer to display tagger, for example `PETSC_VIEWER_STDOUT_WORLD`
