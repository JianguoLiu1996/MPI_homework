# VecSetLayout
set `PetscLayout` describing vector layout 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecSetLayout(Vec x, PetscLayout map)
```
Not Collective


## Input Parameters

- ***x -*** the vector
- ***map -*** the layout





## Note
It is normally only valid to replace the layout with a layout known to be equivalent.
