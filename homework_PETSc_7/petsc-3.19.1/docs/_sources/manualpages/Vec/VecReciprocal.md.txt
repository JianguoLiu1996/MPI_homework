# VecReciprocal
Replaces each component of a vector by its reciprocal. 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecReciprocal(Vec vec)
```
Logically Collective


## Input Parameter

- ***vec -*** the vector



## Output Parameter

- ***vec -*** the vector reciprocal





## Note
Vector entries with value 0.0 are not changed
