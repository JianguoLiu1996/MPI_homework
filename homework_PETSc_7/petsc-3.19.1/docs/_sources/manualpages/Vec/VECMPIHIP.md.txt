# VECMPIHIP
VECMPIHIP = "mpihip" - The basic parallel vector, modified to use HIP 
## Options Database Keys

- ***-vec_type mpihip -*** sets the vector type to `VECMPIHIP` during a call to `VecSetFromOptions()`
