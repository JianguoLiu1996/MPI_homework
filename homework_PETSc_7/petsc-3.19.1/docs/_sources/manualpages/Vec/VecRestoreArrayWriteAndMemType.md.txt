# VecRestoreArrayWriteAndMemType
Restore array obtained with `VecGetArrayWriteAndMemType()` 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecRestoreArrayWriteAndMemType(Vec x, PetscScalar **a)
```
Not Collective; No Fortran Support


## Input Parameters

- ***vec -*** the vector
- ***array -*** the array
