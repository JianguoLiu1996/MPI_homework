# VecNestGetSize
Returns the size of the nest vector. 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecNestGetSize(Vec X, PetscInt *N)
```
Not Collective


## Input Parameter

- ***X  -*** nest vector



## Output Parameter

- ***N -*** number of nested vecs
