# VecViennaCLResetArray
Resets a vector to use its default memory. Call this after the use of VecViennaCLPlaceArray(). 
## Synopsis
```
#include "petscvec.h" 
PETSC_EXTERN PetscErrorCode VecViennaCLResetArray(Vec vin)
```
Not Collective


## Input Parameters

- ***vec -*** the vector
