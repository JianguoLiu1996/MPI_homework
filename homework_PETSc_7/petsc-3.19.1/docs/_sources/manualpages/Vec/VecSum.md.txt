# VecSum
Computes the sum of all the components of a vector. 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecSum(Vec v, PetscScalar *sum)
```
Collective


## Input Parameter

- ***v -*** the vector



## Output Parameter

- ***sum -*** the result





## See Also
 `Vec`, `VecMean()`, `VecNorm()`

beginner
