# VecBoundToCPU
query if a vector is bound to the CPU 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecBoundToCPU(Vec v, PetscBool *flg)
```
Not collective


## Input Parameter

- ***v -*** the vector



## Output Parameter

- ***flg -*** the logical flag
