# VecPow
Replaces each component of a vector by x_i^p 
## Synopsis
```
#include "petscvec.h"  
PetscErrorCode VecPow(Vec v, PetscScalar p)
```
Logically Collective


## Input Parameters

- ***v -*** the vector
- ***p -*** the exponent to use on each element
