# VecTaggerCDFGetMethod

```
#include "petscvec.h" 
PetscErrorCode VecTaggerCDFGetMethod(VecTagger tagger, VecTaggerCDFMethod *method)
```
Logically Collective


## Input Parameter

- ***tagger -*** the `VecTagger` context



## Output Parameter
