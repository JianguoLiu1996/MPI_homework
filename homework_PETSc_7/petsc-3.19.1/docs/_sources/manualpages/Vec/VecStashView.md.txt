# VecStashView
Prints the entries in the vector stash and block stash. 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecStashView(Vec v, PetscViewer viewer)
```
Collective


## Input Parameters

- ***v -*** the vector
- ***viewer -*** the viewer
