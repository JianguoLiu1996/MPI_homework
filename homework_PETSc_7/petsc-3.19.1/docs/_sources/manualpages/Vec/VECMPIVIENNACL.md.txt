# VECMPIVIENNACL
VECMPIVIENNACL = "mpiviennacl" - The basic parallel vector, modified to use ViennaCL 
## Options Database Keys

- ***-vec_type mpiviennacl -*** sets the vector type to VECMPIVIENNACL during a call to VecSetFromOptions()
