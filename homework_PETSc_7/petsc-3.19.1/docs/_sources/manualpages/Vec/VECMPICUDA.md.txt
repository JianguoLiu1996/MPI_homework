# VECMPICUDA
VECMPICUDA = "mpicuda" - The basic parallel vector, modified to use CUDA 
## Options Database Keys

- ***-vec_type mpicuda -*** sets the vector type to `VECMPICUDA` during a call to `VecSetFromOptions()`
