# VecZeroEntries
puts a `0.0` in each element of a vector 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecZeroEntries(Vec vec)
```
Logically Collective


## Input Parameter

- ***vec -*** The vector
