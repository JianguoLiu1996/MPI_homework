# VecShift
Shifts all of the components of a vector by computing `x[i] = x[i] + shift`. 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecShift(Vec v, PetscScalar shift)
```
Logically Collective


## Input Parameters

- ***v -*** the vector
- ***shift -*** the shift
