# VECSEQVIENNACL
VECSEQVIENNACL = "seqviennacl" - The basic sequential vector, modified to use ViennaCL 
## Options Database Keys

- ***-vec_type seqviennacl -*** sets the vector type to VECSEQVIENNACL during a call to VecSetFromOptions()
