# PetscFEGetDimension
Get the dimension of the finite element space on a cell 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEGetDimension(PetscFE fem, PetscInt *dim)
```
Not Collective


## Input Parameter

- ***fe -*** The `PetscFE`



## Output Parameter

- ***dim -*** The dimension
