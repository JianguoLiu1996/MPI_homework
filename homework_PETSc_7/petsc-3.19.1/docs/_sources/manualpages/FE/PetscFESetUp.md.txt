# PetscFESetUp
Construct data structures for the `PetscFE` after the `PetscFEType` has been set 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFESetUp(PetscFE fem)
```
Collective


## Input Parameter

- ***fem -*** the `PetscFE` object to setup
