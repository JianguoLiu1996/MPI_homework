# PetscFEOpenCLGetRealType
Get the scalar type for running on the OpenCL accelerator 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEOpenCLGetRealType(PetscFE fem, PetscDataType *realType)
```

## Input Parameter

- ***fem -*** The `PetscFE`



## Output Parameter

- ***realType -*** The scalar type
