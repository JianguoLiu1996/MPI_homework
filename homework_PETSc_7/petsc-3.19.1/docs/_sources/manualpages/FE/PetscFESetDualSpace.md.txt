# PetscFESetDualSpace
Sets the `PetscDualSpace` used to define the inner product 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFESetDualSpace(PetscFE fem, PetscDualSpace sp)
```
Not Collective


## Input Parameters

- ***fem -*** The `PetscFE` object
- ***sp -*** The `PetscDualSpace` object
