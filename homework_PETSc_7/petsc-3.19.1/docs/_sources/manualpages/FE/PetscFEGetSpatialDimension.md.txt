# PetscFEGetSpatialDimension
Returns the spatial dimension of the element 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEGetSpatialDimension(PetscFE fem, PetscInt *dim)
```
Not Collective


## Input Parameter

- ***fem -*** The `PetscFE` object



## Output Parameter

- ***dim -*** The spatial dimension
