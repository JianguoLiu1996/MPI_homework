# PetscFEOpenCLSetRealType
Set the scalar type for running on the OpenCL accelerator 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEOpenCLSetRealType(PetscFE fem, PetscDataType realType)
```

## Input Parameters

- ***fem      -*** The `PetscFE`
- ***realType -*** The scalar type
