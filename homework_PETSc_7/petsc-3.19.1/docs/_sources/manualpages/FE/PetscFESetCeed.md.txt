# PetscFESetCeed
Set the `Ceed` object to a `PetscFE` 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFESetCeed(PetscFE fe, Ceed ceed)
```
Not Collective


## Input Parameters

- ***fe   -*** The `PetscFE`
- ***ceed -*** The `Ceed` object
