# PetscFEGetDualSpace
Returns the `PetscDualSpace` used to define the inner product for a `PetscFE` 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEGetDualSpace(PetscFE fem, PetscDualSpace *sp)
```
Not Collective


## Input Parameter

- ***fem -*** The `PetscFE` object



## Output Parameter

- ***sp -*** The `PetscDualSpace` object
