# PetscFEGetNumComponents
Returns the number of components in the element 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEGetNumComponents(PetscFE fem, PetscInt *comp)
```
Not Collective


## Input Parameter

- ***fem -*** The `PetscFE` object



## Output Parameter

- ***comp -*** The number of field components
