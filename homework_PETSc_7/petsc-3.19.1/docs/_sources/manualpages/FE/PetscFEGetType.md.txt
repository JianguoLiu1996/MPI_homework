# PetscFEGetType
Gets the `PetscFEType` (as a string) from the `PetscFE` object. 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEGetType(PetscFE fem, PetscFEType *name)
```
Not Collective


## Input Parameter

- ***fem  -*** The `PetscFE`



## Output Parameter

- ***name -*** The `PetscFEType` name
