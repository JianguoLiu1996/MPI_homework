# PetscFESetNumComponents
Sets the number of field components in the element 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFESetNumComponents(PetscFE fem, PetscInt comp)
```
Not Collective


## Input Parameters

- ***fem -*** The `PetscFE` object
- ***comp -*** The number of field components
