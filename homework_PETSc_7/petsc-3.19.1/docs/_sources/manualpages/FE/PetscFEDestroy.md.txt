# PetscFEDestroy
Destroys a `PetscFE` object 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFEDestroy(PetscFE *fem)
```
Collective


## Input Parameter

- ***fem -*** the `PetscFE` object to destroy





## See Also
 `PetscFE`, `PetscFEView()`

beginner
