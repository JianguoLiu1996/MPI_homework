# PetscCommDestroy
Frees communicator obtained with `PetscCommDuplicate()`. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscCommDestroy(MPI_Comm *comm)
```
Collective


## Input Parameter

- ***comm -*** the communicator to free
