# PetscBagGetName
Gets the name of a bag of values 
## Synopsis
```
#include "petscbag.h"   
PetscErrorCode PetscBagGetName(PetscBag bag, char **name)
```
Not Collective




## Input Parameter

- ***bag -*** the bag of values



## Output Parameter

- ***name -*** the name assigned to the bag
