# PetscMallocLogRequestedSizeGet
Whether to log the requested or aligned memory size 
## Synopsis
```
#include "petscsys.h" 
PetscErrorCode PetscMallocLogRequestedSizeGet(PetscBool *flg)
```
Not Collective


## Output Parameter

- ***flg -*** `PETSC_TRUE` if we log the requested memory size
