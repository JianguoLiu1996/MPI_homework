# PETSC_MPI_THREAD_REQUIRED
the required threading support used if PETSc initializes MPI with `MPI_Init_thread()`. 


## Notes
By default `PETSC_MPI_THREAD_REQUIRED` equals `MPI_THREAD_FUNNELED`.


## See Also
