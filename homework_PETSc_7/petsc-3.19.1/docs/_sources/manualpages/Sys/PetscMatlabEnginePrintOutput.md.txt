# PetscMatlabEnginePrintOutput
prints the output from MATLAB to an ASCII file 
## Synopsis
```
#include "petscmatlab.h"  
PetscErrorCode PetscMatlabEnginePrintOutput(PetscMatlabEngine mengine, FILE *fd)
```
Collective


## Input Parameters

- ***mengine -*** the MATLAB engine
- ***fd -*** the file
