# PetscDLClose
closes a dynamic library 
## Synopsis
```
PetscErrorCode PetscDLClose(PetscDLHandle *handle)
```
Not Collective


## Input Parameter

- ***handle -*** the handle for the library obtained with `PetscDLOpen()`
