# PetscBagGetNames
Get the names of all entries in the bag 
## Synopsis
```
#include "petscbag.h"   
PetscErrorCode PetscBagGetNames(PetscBag bag, const char *names[])
```
Not Collective


## Input Parameter

- ***bag   -*** the bag of values



## Output Parameter

- ***names -*** array of char pointers for names
