# PetscObjectSetUp
Sets up the internal data structures for the later use. 
## Synopsis
```
#include "petscsys.h"    
PetscErrorCode PetscObjectSetUp(PetscObject obj)
```
Collective


## Input Parameter

- ***obj -*** the `PetscObject`





## Note
This does nothing at present.
