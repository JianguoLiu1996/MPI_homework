# PetscFPTrapPop
push a floating point trapping mode, to be restored using `PetscFPTrapPop()` 
## Synopsis
```
#include "petscsys.h"  
PetscErrorCode PetscFPTrapPop(void)
```
Not Collective




## See Also
 `PetscFPTrapPush()`, `PetscSetFPTrap()`, `PetscDetermineInitialFPTrap()`
