# PetscSortReverseInt
Sorts an array of integers in place in decreasing order. 
## Synopsis
```
#include "petscsys.h"  
PetscErrorCode PetscSortReverseInt(PetscInt n, PetscInt X[])
```
Not Collective


## Input Parameters

- ***n  -*** number of values
- ***X  -*** array of integers





## See Also
 `PetscIntSortSemiOrdered()`, `PetscSortInt()`, `PetscSortIntWithPermutation()`

intermediate
