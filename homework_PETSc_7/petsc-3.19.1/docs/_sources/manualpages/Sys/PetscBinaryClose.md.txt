# PetscBinaryClose
Closes a PETSc binary file. 
## Synopsis
```
PetscErrorCode PetscBinaryClose(int fd)
```
Not Collective


## Output Parameter

- ***fd -*** the file





## See Also
 `PetscBinaryRead()`, `PetscBinaryWrite()`, `PetscBinaryOpen()`, `PetscBinarySynchronizedWrite()`, `PetscBinarySynchronizedRead()`,
`PetscBinarySynchronizedSeek()`
