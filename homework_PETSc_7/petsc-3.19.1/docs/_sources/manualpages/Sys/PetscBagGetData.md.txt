# PetscBagGetData
Gives back the user - access to memory that can be used for storing user-data-structure 
## Synopsis
```
#include "petscbag.h"   
PetscErrorCode PetscBagGetData(PetscBag bag, void **data)
```
Not Collective


## Input Parameter

- ***bag -*** the bag of values



## Output Parameter

- ***data -*** pointer to memory that will have user-data-structure, this can be cast to a pointer of the type the C struct used in
defining the bag
