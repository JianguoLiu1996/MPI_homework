# PetscPopErrorHandler
Removes the latest error handler that was pushed with `PetscPushErrorHandler()`. 
## Synopsis
```
#include "petscsys.h" 
PetscErrorCode PetscPopErrorHandler(void)
```
Not Collective




## See Also
 `PetscPushErrorHandler()`

intermediate
