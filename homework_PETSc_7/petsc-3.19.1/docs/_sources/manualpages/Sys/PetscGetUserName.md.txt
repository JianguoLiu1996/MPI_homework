# PetscGetUserName
Returns the name of the user. 
## Synopsis
```
PetscErrorCode PetscGetUserName(char name[], size_t nlen)
```
Not Collective


## Input Parameter
nlen - length of name


## Output Parameter

- ***name -*** contains user name.  Must be long enough to hold the name
