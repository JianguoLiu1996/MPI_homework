# PetscMatlabEngineDestroy
Shuts down a MATLAB engine. 
## Synopsis
```
#include "petscmatlab.h"  
PetscErrorCode PetscMatlabEngineDestroy(PetscMatlabEngine *v)
```
Collective


## Input Parameter

- ***e  -*** the engine
