# PetscObjectDestroyOptionsHandlers
Destroys all the option handlers attached to an object 
## Synopsis
```
#include "petscsys.h"    
PetscErrorCode PetscObjectDestroyOptionsHandlers(PetscObject obj)
```
Not Collective


## Input Parameter

- ***obj -*** the PETSc object
