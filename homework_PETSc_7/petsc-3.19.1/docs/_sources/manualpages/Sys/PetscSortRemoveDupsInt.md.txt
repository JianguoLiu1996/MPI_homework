# PetscSortRemoveDupsInt
Sorts an array of `PetscInt` in place in increasing order removes all duplicate entries 
## Synopsis
```
#include "petscsys.h"  
PetscErrorCode PetscSortRemoveDupsInt(PetscInt *n, PetscInt X[])
```
Not Collective


## Input Parameters

- ***n  -*** number of values
- ***X  -*** array of integers



## Output Parameter

- ***n -*** number of non-redundant values
