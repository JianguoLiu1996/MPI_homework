# PetscSortInt64
Sorts an array of `PetscInt64` in place in increasing order. 
## Synopsis
```
#include "petscsys.h"  
PetscErrorCode PetscSortInt64(PetscInt n, PetscInt64 X[])
```
Not Collective


## Input Parameters

- ***n  -*** number of values
- ***X  -*** array of integers



## Notes
This function sorts `PetscCount`s assumed to be in completely random order

 `PetscSortInt()`
