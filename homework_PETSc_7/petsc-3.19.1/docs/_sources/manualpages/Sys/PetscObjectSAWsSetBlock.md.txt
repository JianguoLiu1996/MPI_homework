# PetscObjectSAWsSetBlock
Sets whether an object will block at `PetscObjectSAWsBlock()` 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscObjectSAWsSetBlock(PetscObject obj, PetscBool flg)
```
Collective


## Input Parameters

- ***obj -*** the PETSc variable
- ***flg -*** whether it should block
