# PetscSortedRemoveDupsInt
Removes all duplicate entries of a sorted `PetscInt` array 
## Synopsis
```
#include "petscsys.h"  
PetscErrorCode PetscSortedRemoveDupsInt(PetscInt *n, PetscInt X[])
```
Not Collective


## Input Parameters

- ***n  -*** number of values
- ***X  -*** sorted array of integers



## Output Parameter

- ***n -*** number of non-redundant values

 `PetscSortInt()`

intermediate
