# PetscAbs
Returns the absolute value of a number 
## Synopsis
```
#include <petscmath.h>
type PetscAbs(type v)
```
Not Collective


## Input Parameter

- ***v -*** the number





## Note
The type can be integer or real floating point value, but cannot be complex


## See Also
 `PetscAbsInt()`, `PetscAbsReal()`, `PetscAbsScalar()`, `PetscSign()`

## Level
beginner
