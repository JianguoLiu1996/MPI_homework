# PetscAddrAlign
Rounds up an address to `PETSC_MEMALIGN` alignment 
## Synopsis
```
#include <petscsys.h>
void *PetscAddrAlign(void *addr)
```
Not Collective


## Input Parameter

- ***addr -*** address to align (any pointer type)





## See Also
 `PetscMallocAlign()`

developer
