# PetscSleep
Sleeps some number of seconds. 
## Synopsis
```
#include "petscsys.h"    
PetscErrorCode PetscSleep(PetscReal s)
```
Not Collective


## Input Parameter

- ***s -*** number of seconds to sleep





## Note
If s is negative waits for keyboard input


## Level
intermediate
