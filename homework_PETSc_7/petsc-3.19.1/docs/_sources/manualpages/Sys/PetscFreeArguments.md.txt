# PetscFreeArguments
Frees the memory obtained with `PetscGetArguments()` 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscFreeArguments(char **args)
```
Not Collective


## Output Parameter

- ***args -*** the command line arguments
