# PetscSizeT
datatype used to represent sizes in memory (like `size_t`) 


## Notes
This is equivalent to `size_t`, but defined for consistency with Fortran, which lacks a native equivalent of `size_t`.


## See Also
 `PetscInt`, `PetscInt64`, `PetscCount`
