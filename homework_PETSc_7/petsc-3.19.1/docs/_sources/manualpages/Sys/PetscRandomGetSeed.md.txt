# PetscRandomGetSeed
Gets the random seed. 
## Synopsis
```
#include "petscsys.h" 
PetscErrorCode PetscRandomGetSeed(PetscRandom r, unsigned long *seed)
```
Not collective


## Input Parameter

- ***r -*** The random number generator context



## Output Parameter

- ***seed -*** The random seed
