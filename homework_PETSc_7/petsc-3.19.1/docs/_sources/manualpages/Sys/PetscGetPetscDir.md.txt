# PetscGetPetscDir
Gets the directory PETSc is installed in 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscGetPetscDir(const char *dir[])
```
Not Collective; No Fortran Support


## Output Parameter

- ***dir -*** the directory

developer
