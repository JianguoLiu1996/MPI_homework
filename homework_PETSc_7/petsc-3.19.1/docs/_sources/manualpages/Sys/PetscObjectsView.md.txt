# PetscObjectsView
Prints the currently existing objects. 
## Synopsis
```
#include "petscsys.h"    
PetscErrorCode PetscObjectsView(PetscViewer viewer)
```
Logically Collective


## Input Parameter

- ***viewer -*** must be an `PETSCVIEWERASCII` viewer





## Level
advanced
