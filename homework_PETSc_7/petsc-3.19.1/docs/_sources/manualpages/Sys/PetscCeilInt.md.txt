# PetscCeilInt
Returns the ceiling of the quotation of two positive integers 
## Synopsis
```
#include <petscmath.h>
PetscInt PetscCeilInt(PetscInt x,PetscInt y)
```
Not Collective


## Input Parameters

- ***x -*** the numerator
- ***y -*** the denominator
