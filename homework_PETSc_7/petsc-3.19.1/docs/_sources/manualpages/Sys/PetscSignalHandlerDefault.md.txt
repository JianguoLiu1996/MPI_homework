# PetscSignalHandlerDefault
Default signal handler. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscSignalHandlerDefault(int sig, void *ptr)
```
Not Collective


## Input Parameters

- ***sig -*** signal value
- ***ptr -*** unused pointer





## Developer Note
This does not call `PetscError()`, handles the entire error process directly


## See Also
 `PetscPushSignalHandler()`

advanced
