# PetscRMTree
delete a directory and all of its children 
## Synopsis
```
PetscErrorCode PetscRMTree(const char dir[])
```

## Input Parameter

- ***dir -*** the name of the directory





## See Also
 `PetscMkdtemp()`, `PetscMkdir()`

advanced
