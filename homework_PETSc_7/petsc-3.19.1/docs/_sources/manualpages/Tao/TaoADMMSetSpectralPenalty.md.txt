# TaoADMMSetSpectralPenalty
Set the spectral penalty (mu) value 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoADMMSetSpectralPenalty(Tao tao, PetscReal mu)
```
Collective


## Input Parameters

- ***tao -*** the `Tao` solver context
- ***mu -*** spectral penalty
