# TaoADMMGetDualVector
Returns the dual vector associated with the current `TAOADMM` state 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoADMMGetDualVector(Tao tao, Vec *Y)
```
Not Collective


## Input Parameter

- ***tao -*** the `Tao` context



## Output Parameter

- ***Y -*** the current solution

 `TAOADMM`
