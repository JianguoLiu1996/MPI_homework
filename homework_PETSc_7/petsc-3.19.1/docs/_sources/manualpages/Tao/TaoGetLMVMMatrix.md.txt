# TaoGetLMVMMatrix
Returns a pointer to the internal LMVM matrix. Valid only for quasi-Newton family of methods. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetLMVMMatrix(Tao tao, Mat *B)
```

## Input Parameter

- ***tao -*** `Tao` solver context



## Output Parameter

- ***B -*** LMVM matrix
