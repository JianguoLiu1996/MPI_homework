# TaoAddLineSearchCounts
Adds the number of function evaluations spent in the line search to the running total. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoAddLineSearchCounts(Tao tao)
```

## Input Parameters

- ***tao -*** the `Tao` solver
