# TaoGetApplicationContext
Gets the user-defined context for a `Tao` solver 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetApplicationContext(Tao tao, void *usrP)
```
Not Collective


## Input Parameter

- ***tao  -*** the `Tao` context



## Output Parameter

- ***usrP -*** user context
