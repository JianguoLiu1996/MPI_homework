# TaoGradientNorm
Compute the norm using the `NormType`, the user has selected 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGradientNorm(Tao tao, Vec gradient, NormType type, PetscReal *gnorm)
```
Collective


## Input Parameters

- ***tao      -*** the `Tao` context
- ***gradient -*** the gradient to be computed
- ***norm     -*** the norm type



## Output Parameter

- ***gnorm    -*** the gradient norm
