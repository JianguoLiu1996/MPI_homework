# TaoGetDualVariables
Gets the dual vectors 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetDualVariables(Tao tao, Vec *DE, Vec *DI)
```
Collective


## Input Parameter

- ***tao -*** the `Tao` context



## Output Parameters

- ***DE -*** dual variable vector for the lower bounds
- ***DI -*** dual variable vector for the upper bounds
