# TaoSetGradientNorm
Sets the matrix used to define the norm that measures the size of the gradient. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoSetGradientNorm(Tao tao, Mat M)
```
Collective


## Input Parameters

- ***tao  -*** the `Tao` context
- ***M    -*** matrix that defines the norm
