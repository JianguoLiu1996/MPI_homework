# TaoALMMSetType
Determine the augmented Lagrangian formulation type for the subproblem. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoALMMSetType(Tao tao, TaoALMMType type)
```

## Input Parameters

- ***tao -*** the `Tao` context for the `TAOALMM` solver
- ***type -*** augmented Lagragrangian type
