# TaoALMMGetSubsolver
Retrieve the subsolver being used by `TAOALMM`. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoALMMGetSubsolver(Tao tao, Tao *subsolver)
```

## Input Parameter

- ***tao -*** the `Tao` context for the `TAOALMM` solver



## Output Parameter

- ***subsolver -*** the `Tao` context for the subsolver
