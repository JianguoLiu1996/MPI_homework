# TaoGetMaximumIterations
Gets a maximum number of iterates that will be used 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetMaximumIterations(Tao tao, PetscInt *maxits)
```
Not Collective


## Input Parameter

- ***tao -*** the `Tao` solver context



## Output Parameter

- ***maxits -*** the maximum number of iterates
