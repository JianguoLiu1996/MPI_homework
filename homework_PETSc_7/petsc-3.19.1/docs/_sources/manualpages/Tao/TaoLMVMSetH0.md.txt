# TaoLMVMSetH0
Set the initial Hessian for the QN approximation 
## Synopsis
```
#include "petsctaolinesearch.h" 
PetscErrorCode TaoLMVMSetH0(Tao tao, Mat H0)
```

## Input Parameters

- ***tao  -*** the `Tao` solver context
- ***H0 -*** `Mat` object for the initial Hessian
