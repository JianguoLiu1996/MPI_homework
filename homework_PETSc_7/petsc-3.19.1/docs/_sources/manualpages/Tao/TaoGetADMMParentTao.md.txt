# TaoGetADMMParentTao
Gets pointer to parent `TAOADMM`, used by inner subsolver. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetADMMParentTao(Tao tao, Tao *admm_tao)
```
Collective


## Input Parameter

- ***tao -*** the `Tao` context



## Output Parameter

- ***admm_tao -*** the parent `Tao` context
