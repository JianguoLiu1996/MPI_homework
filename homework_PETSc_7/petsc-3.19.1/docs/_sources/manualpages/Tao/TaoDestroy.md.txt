# TaoDestroy
Destroys the `Tao` context that was created with `TaoCreate()` 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoDestroy(Tao *tao)
```
Collective


## Input Parameter

- ***tao -*** the `Tao` context





## See Also
 [](chapter_tao), `Tao`, `TaoCreate()`, `TaoSolve()`
