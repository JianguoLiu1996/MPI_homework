# TaoSetTotalIterationNumber
Sets the current total iteration number. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoSetTotalIterationNumber(Tao tao, PetscInt iter)
```
Logically Collective


## Input Parameters

- ***tao -*** the `Tao` context
- ***iter -*** the iteration number
