# TaoADMMGetSpectralPenalty
Get the spectral penalty (mu) value 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoADMMGetSpectralPenalty(Tao tao, PetscReal *mu)
```
Collective


## Input Parameter

- ***tao -*** the `Tao` solver context



## Output Parameter

- ***mu -*** spectral penalty
