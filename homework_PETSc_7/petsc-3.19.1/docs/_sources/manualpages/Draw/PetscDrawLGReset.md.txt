# PetscDrawLGReset
Clears line graph to allow for reuse with new data. 
## Synopsis
```
#include "petscdraw.h"  
PetscErrorCode PetscDrawLGReset(PetscDrawLG lg)
```
Logically Collective


## Input Parameter

- ***lg -*** the line graph context.





## See Also
 `PetscDrawLG`, `PetscDrawLGCreate()`

## Level
intermediate
