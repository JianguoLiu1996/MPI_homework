# PetscDrawPopCurrentPoint
Pops a current draw point (discarding it) 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawPopCurrentPoint(PetscDraw draw)
```
Not Collective


## Input Parameter

- ***draw -*** the drawing context
