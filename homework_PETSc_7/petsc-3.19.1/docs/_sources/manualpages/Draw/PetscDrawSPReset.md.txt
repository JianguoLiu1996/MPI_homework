# PetscDrawSPReset
Clears scatter plot to allow for reuse with new data. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
PetscErrorCode PetscDrawSPReset(PetscDrawSP sp)
```
Not Collective


## Input Parameter

- ***sp -*** the scatter plot context.
