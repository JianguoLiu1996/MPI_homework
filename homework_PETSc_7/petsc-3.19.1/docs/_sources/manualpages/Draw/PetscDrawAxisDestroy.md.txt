# PetscDrawAxisDestroy
Frees the space used by an axis structure. 
## Synopsis
```
#include "petscdraw.h"  
PetscErrorCode PetscDrawAxisDestroy(PetscDrawAxis *axis)
```
Collective


## Input Parameter

- ***axis -*** the axis context
