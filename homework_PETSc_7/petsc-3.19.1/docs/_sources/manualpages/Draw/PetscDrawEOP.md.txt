# PetscDrawEOP
Ends a page or frame on the selected graphical device. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawEOP(PetscDraw draw)
```
Logically Collective


## Input Parameter

- ***draw -*** the drawing context





## See Also
 `PetscDrawBOP()`, `PetscDrawClear()`

advanced
