# PetscDrawSPDestroy
Frees all space taken up by scatter plot data structure. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
PetscErrorCode PetscDrawSPDestroy(PetscDrawSP *sp)
```
Collective


## Input Parameter

- ***sp -*** the scatter plot context
