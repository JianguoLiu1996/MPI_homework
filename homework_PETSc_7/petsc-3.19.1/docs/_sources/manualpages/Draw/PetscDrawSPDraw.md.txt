# PetscDrawSPDraw
Redraws a scatter plot. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
PetscErrorCode PetscDrawSPDraw(PetscDrawSP sp, PetscBool clear)
```
Collective


## Input Parameters

- ***sp -*** the scatter plot context
- ***clear -*** clear the window before drawing the new plot
