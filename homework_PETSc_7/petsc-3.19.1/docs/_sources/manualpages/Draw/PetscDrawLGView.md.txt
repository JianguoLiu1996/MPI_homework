# PetscDrawLGView
Prints a line graph. 
## Synopsis
```
#include "petscdraw.h"  
PetscErrorCode PetscDrawLGView(PetscDrawLG lg, PetscViewer viewer)
```
Collective


## Input Parameter

- ***lg -*** the line graph context
