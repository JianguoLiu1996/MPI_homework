# PetscDrawOpenNull
Opens a null drawing context. All draw commands to it are ignored. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawOpenNull(MPI_Comm comm, PetscDraw *win)
```

## Input Parameter

- ***comm -*** MPI communicator



## Output Parameter

- ***draw -*** the drawing context
