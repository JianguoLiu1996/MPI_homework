# PetscDrawSetDoubleBuffer
Sets a window to be double buffered. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawSetDoubleBuffer(PetscDraw draw)
```
Logically Collective


## Input Parameter

- ***draw -*** the drawing context





## See Also
 `PetscDraw`, `PetscDrawOpenX()`, `PetscDrawCreate()`
