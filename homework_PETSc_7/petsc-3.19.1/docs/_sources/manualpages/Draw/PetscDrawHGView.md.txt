# PetscDrawHGView
Prints the histogram information to a viewer 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawHGView(PetscDrawHG hist, PetscViewer viewer)
```
Not Collective


## Input Parameter

- ***hist -*** The histogram context
