# PetscDrawDestroy
Deletes a draw context. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawDestroy(PetscDraw *draw)
```
Collective


## Input Parameter

- ***draw -*** the drawing context





## See Also
 `PetscDraw`, `PetscDrawCreate()`
