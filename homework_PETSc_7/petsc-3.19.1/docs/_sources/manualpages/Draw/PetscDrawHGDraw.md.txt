# PetscDrawHGDraw
Redraws a histogram. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawHGDraw(PetscDrawHG hist)
```
Collective


## Input Parameter

- ***hist -*** The histogram context
