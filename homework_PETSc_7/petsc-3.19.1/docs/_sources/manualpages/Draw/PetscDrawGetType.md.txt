# PetscDrawGetType
Gets the `PetscDraw` type as a string from the `PetscDraw` object. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawGetType(PetscDraw draw, PetscDrawType *type)
```
Not Collective


## Input Parameter

- ***draw -*** Krylov context



## Output Parameter

- ***name -*** name of PetscDraw method
