# PetscDrawHGAddValue
Adds another value to the histogram. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawHGAddValue(PetscDrawHG hist, PetscReal value)
```
Logically Collective


## Input Parameters

- ***hist  -*** The histogram
- ***value -*** The value
