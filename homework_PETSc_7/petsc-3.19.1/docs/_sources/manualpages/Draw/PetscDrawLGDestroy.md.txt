# PetscDrawLGDestroy
Frees all space taken up by line graph data structure. 
## Synopsis
```
#include "petscdraw.h"  
PetscErrorCode PetscDrawLGDestroy(PetscDrawLG *lg)
```
Collective


## Input Parameter

- ***lg -*** the line graph context





## See Also
 `PetscDrawLG`, `PetscDrawLGCreate()`

## Level
intermediate
