# PetscDrawIsNull
Returns `PETSC_TRUE` if draw is a null draw object. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawIsNull(PetscDraw draw, PetscBool *yes)
```
Not Collective


## Input Parameter

- ***draw -*** the draw context



## Output Parameter

- ***yes -*** `PETSC_TRUE` if it is a null draw object; otherwise `PETSC_FALSE`
