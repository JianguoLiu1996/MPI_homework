# PetscDrawGetWindowSize
Gets the size of the window. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawGetWindowSize(PetscDraw draw, int *w, int *h)
```
Not Collective


## Input Parameter

- ***draw -*** the window



## Output Parameters

- ***w -*** the window width
- ***h -*** the window height





## See Also
 `PetscDraw`, `PetscDrawResizeWindow()`, `PetscDrawCheckResizedWindow()`

intermediate
