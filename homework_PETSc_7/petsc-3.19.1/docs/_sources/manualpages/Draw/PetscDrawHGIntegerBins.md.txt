# PetscDrawHGIntegerBins
Turns on integer width bins 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawHGIntegerBins(PetscDrawHG hist, PetscBool ints)
```
Not Collective


## Input Parameters

- ***hist -*** The histogram context
- ***ints -*** Flag for integer width bins
