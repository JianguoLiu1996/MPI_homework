# PetscDrawSave
Saves a drawn image 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawSave(PetscDraw draw)
```
Collective


## Input Parameter

- ***draw -*** the drawing context





## Note
this is not normally called by the user.


## See Also
 `PetscDraw`, `PetscDrawSetSave()`

advanced
