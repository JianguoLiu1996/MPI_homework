# PetscDrawHGDestroy
Frees all space taken up by histogram data structure. 
## Synopsis
```
#include "petscdraw.h" 
#include "petscsys.h" 
#include "petscviewer.h" 
PetscErrorCode PetscDrawHGDestroy(PetscDrawHG *hist)
```
Collective


## Input Parameter

- ***hist -*** The histogram context
