# PetscDrawLGDraw
Redraws a line graph. 
## Synopsis
```
#include "petscdraw.h"  
PetscErrorCode PetscDrawLGDraw(PetscDrawLG lg)
```
Collective


## Input Parameter

- ***lg -*** the line graph context
