# PetscDrawClear
Clears graphical output. All processors must call this routine. Does not return until the draw in context is clear. 
## Synopsis
```
#include "petscdraw.h" 
PetscErrorCode PetscDrawClear(PetscDraw draw)
```
Collective


## Input Parameter

- ***draw -*** the drawing context





## Level
intermediate
