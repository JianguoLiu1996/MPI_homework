# DMMoabGetOffset
Get the local offset for the global vector 
## Synopsis
```
#include "petscdmmoab.h"   
PetscErrorCode DMMoabGetOffset(DM dm, PetscInt *offset)
```
Collective


## Input Parameter

- ***dm -*** The DMMoab object being set



## Output Parameter

- ***offset -*** The local offset for the global vector
