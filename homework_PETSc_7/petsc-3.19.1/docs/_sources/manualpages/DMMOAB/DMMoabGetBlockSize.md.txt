# DMMoabGetBlockSize
Get the block size used with this DMMoab 
## Synopsis
```
#include "petscdmmoab.h"   
PetscErrorCode DMMoabGetBlockSize(DM dm, PetscInt *bs)
```
Collective


## Input Parameter

- ***dm -*** The DMMoab object being set



## Output Parameter

- ***bs -*** The block size used with this DMMoab
