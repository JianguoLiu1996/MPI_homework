# DMMoabCreate
Creates a DMMoab object, which encapsulates a moab instance 
## Synopsis
```
#include "petscdmmoab.h"   
PetscErrorCode DMMoabCreate(MPI_Comm comm, DM *dmb)
```
Collective


## Input Parameter

- ***comm -*** The communicator for the DMMoab object



## Output Parameter

- ***dmb  -*** The DMMoab object
