# DMMoabSetBlockSize
Set the block size used with this DMMoab 
## Synopsis
```
#include "petscdmmoab.h"   
PetscErrorCode DMMoabSetBlockSize(DM dm, PetscInt bs)
```
Collective


## Input Parameters

- ***dm -*** The DMMoab object being set
- ***bs -*** The block size used with this DMMoab
