# DMMoabGetDimension
Get the dimension of the DM Mesh 
## Synopsis
```
#include "petscdmmoab.h"   
PetscErrorCode DMMoabGetDimension(DM dm, PetscInt *dim)
```
Collective


## Input Parameter

- ***dm -*** The DMMoab object



## Output Parameter

- ***dim -*** The dimension of DM
