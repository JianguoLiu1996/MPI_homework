# PetscDualSpaceLagrangeGetUseMoments

## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceLagrangeGetUseMoments(PetscDualSpace sp, PetscBool *useMoments)
```
Not Collective


## Input Parameter

- ***sp -*** The `PetscDualSpace`



## Output Parameter
