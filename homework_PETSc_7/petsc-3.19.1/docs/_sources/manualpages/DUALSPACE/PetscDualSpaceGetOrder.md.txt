# PetscDualSpaceGetOrder
Get the order of the dual space 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceGetOrder(PetscDualSpace sp, PetscInt *order)
```
Not Collective


## Input Parameter

- ***sp -*** The `PetscDualSpace`



## Output Parameter

- ***order -*** The order
