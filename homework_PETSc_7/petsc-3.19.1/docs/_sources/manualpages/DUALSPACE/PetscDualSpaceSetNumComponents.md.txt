# PetscDualSpaceSetNumComponents
Set the number of components for this space 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceSetNumComponents(PetscDualSpace sp, PetscInt Nc)
```

## Input Parameters

- ***sp -*** The `PetscDualSpace`
- ***order -*** The number of components
