# PetscDualSpaceGetDimension
Get the dimension of the dual space, i.e. the number of basis functionals 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceGetDimension(PetscDualSpace sp, PetscInt *dim)
```
Not Collective


## Input Parameter

- ***sp -*** The `PetscDualSpace`



## Output Parameter

- ***dim -*** The dimension
