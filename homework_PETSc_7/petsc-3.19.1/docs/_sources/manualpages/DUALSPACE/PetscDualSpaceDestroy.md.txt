# PetscDualSpaceDestroy
Destroys a `PetscDualSpace` object 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceDestroy(PetscDualSpace *sp)
```
Collective


## Input Parameter

- ***sp -*** the `PetscDualSpace` object to destroy
