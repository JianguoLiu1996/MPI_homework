# PetscLogMPEDump
Dumps the MPE logging info to file for later use with Jumpshot. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscLogMPEDump(const char sname[])
```
Collective over `PETSC_COMM_WORLD`
