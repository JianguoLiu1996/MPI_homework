# PetscLogViewFromOptions
Processes command line options to determine if/how a `PetscLog` is to be viewed. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscLogViewFromOptions(void)
```
Collective on `PETSC_COMM_WORLD`
