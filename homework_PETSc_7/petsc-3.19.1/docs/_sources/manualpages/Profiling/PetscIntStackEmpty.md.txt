# PetscIntStackEmpty
This function determines whether any items have been pushed. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscIntStackEmpty(PetscIntStack stack, PetscBool *empty)
```
Not Collective


## Input Parameter

- ***stack -*** The stack



## Output Parameter

- ***empty -*** `PETSC_TRUE` if the stack is empty
