# PetscIntStackPush
This function pushes an integer on the stack. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscIntStackPush(PetscIntStack stack, int item)
```
Not Collective


## Input Parameters

- ***stack -*** The stack
- ***item  -*** The integer to push
