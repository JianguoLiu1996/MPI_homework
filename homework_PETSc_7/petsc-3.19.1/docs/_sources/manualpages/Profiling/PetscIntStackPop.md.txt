# PetscIntStackPop
This function pops an integer from the stack. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscIntStackPop(PetscIntStack stack, int *item)
```
Not Collective


## Input Parameter

- ***stack -*** The stack



## Output Parameter

- ***item  -*** The integer popped
