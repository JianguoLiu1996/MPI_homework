# PetscInfoAllow
Enables/disables `PetscInfo()` messages 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscInfoAllow(PetscBool flag)
```
Not Collective


## Input Parameter

- ***flag -*** `PETSC_TRUE` or `PETSC_FALSE`
