# PetscIntStackTop
This function returns the top of the stack. 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscIntStackTop(PetscIntStack stack, int *top)
```
Not Collective


## Input Parameter

- ***stack -*** The stack



## Output Parameter

- ***top -*** The integer on top of the stack
