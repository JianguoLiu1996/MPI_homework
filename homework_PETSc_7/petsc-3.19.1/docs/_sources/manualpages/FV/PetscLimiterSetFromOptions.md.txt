# PetscLimiterSetFromOptions
sets parameters in a `PetscLimiter` from the options database 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscLimiterSetFromOptions(PetscLimiter lim)
```
Collective


## Input Parameter

- ***lim -*** the `PetscLimiter` object to set options for
