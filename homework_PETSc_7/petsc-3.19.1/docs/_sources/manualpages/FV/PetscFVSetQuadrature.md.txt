# PetscFVSetQuadrature
Set the `PetscQuadrature` object for a `PetscFV` 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVSetQuadrature(PetscFV fvm, PetscQuadrature q)
```
Logically Collective


## Input Parameters

- ***fvm -*** the `PetscFV` object
- ***q -*** The `PetscQuadrature`
