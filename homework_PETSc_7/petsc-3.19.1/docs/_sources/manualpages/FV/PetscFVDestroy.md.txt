# PetscFVDestroy
Destroys a `PetscFV` object 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVDestroy(PetscFV *fvm)
```
Collective


## Input Parameter

- ***fvm -*** the `PetscFV` object to destroy
