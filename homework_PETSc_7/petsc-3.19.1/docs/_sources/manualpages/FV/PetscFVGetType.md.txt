# PetscFVGetType
Gets the `PetscFVType` (as a string) from a `PetscFV`. 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVGetType(PetscFV fvm, PetscFVType *name)
```
Not Collective


## Input Parameter

- ***fvm  -*** The `PetscFV`



## Output Parameter

- ***name -*** The `PetscFVType` name
