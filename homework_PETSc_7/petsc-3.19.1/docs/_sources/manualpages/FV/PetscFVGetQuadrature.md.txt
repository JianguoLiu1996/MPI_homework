# PetscFVGetQuadrature
Get the `PetscQuadrature` from a `PetscFV` 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVGetQuadrature(PetscFV fvm, PetscQuadrature *q)
```
Not Collective


## Input Parameter

- ***fvm -*** the `PetscFV` object



## Output Parameter

- ***lim -*** The `PetscQuadrature`
