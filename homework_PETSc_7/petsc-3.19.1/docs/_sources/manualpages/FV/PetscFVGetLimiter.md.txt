# PetscFVGetLimiter
Get the `PetscLimiter` object from the `PetscFV` 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVGetLimiter(PetscFV fvm, PetscLimiter *lim)
```
Not Collective


## Input Parameter

- ***fvm -*** the `PetscFV` object



## Output Parameter

- ***lim -*** The `PetscLimiter`





## See Also
 `PetscFV`, `PetscLimiter`, `PetscFVSetLimiter()`
