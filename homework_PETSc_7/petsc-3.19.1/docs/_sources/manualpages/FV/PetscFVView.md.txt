# PetscFVView
Views a `PetscFV` 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVView(PetscFV fvm, PetscViewer v)
```
Collective


## Input Parameters

- ***fvm -*** the `PetscFV` object to view
- ***v   -*** the viewer
