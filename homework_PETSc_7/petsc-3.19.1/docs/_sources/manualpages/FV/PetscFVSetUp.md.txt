# PetscFVSetUp
Setup the data structures for the `PetscFV` based on the `PetscFVType` provided by `PetscFVSetType()` 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscFVSetUp(PetscFV fvm)
```
Collective


## Input Parameter

- ***fvm -*** the `PetscFV` object to setup
