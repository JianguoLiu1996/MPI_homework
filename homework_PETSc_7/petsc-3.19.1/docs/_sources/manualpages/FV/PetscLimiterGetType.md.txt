# PetscLimiterGetType
Gets the `PetscLimiterType` name (as a string) from the `PetscLimiter`. 
## Synopsis
```
#include "petscfv.h" 
PetscErrorCode PetscLimiterGetType(PetscLimiter lim, PetscLimiterType *name)
```
Not Collective


## Input Parameter

- ***lim  -*** The `PetscLimiter`



## Output Parameter

- ***name -*** The `PetscLimiterType`
