# KSPResetViewers
Resets all the viewers set from the options database during `KSPSetFromOptions()` 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPResetViewers(KSP ksp)
```
Collective


## Input Parameter

- ***ksp -*** iterative context obtained from `KSPCreate()`
