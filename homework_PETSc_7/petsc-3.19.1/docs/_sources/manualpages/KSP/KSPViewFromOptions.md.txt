# KSPViewFromOptions
View a `KSP` object based on values in the options database 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPViewFromOptions(KSP A, PetscObject obj, const char name[])
```
Collective


## Input Parameters

- ***A -*** Krylov solver context
