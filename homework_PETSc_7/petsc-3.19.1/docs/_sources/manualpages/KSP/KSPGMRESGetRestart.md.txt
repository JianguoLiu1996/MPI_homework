# KSPGMRESGetRestart
Gets number of iterations at which `KSPGMRES`, `KSPFGMRES` and `KSPLGMRES` restarts. 
## Synopsis
```
#include "petscksp.h"  
PetscErrorCode KSPGMRESGetRestart(KSP ksp, PetscInt *restart)
```
Not Collective


## Input Parameter

- ***ksp -*** the Krylov space context



## Output Parameter

- ***restart -*** integer restart value
