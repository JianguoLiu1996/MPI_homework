# KSPMonitorErrorDrawLGCreate
Creates the context for the error and preconditioned residual plotter `KSPMonitorErrorDrawLG()` 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPMonitorErrorDrawLGCreate(PetscViewer viewer, PetscViewerFormat format, void *ctx, PetscViewerAndFormat **vf)
```
Collective


## Input Parameters

- ***viewer -*** The `PetscViewer`
- ***format -*** The viewer format
- ***ctx    -*** An optional user context



## Output Parameter

- ***vf    -*** The viewer context
