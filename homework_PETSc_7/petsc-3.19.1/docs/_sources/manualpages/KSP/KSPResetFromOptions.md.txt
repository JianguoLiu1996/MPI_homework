# KSPResetFromOptions
Sets `KSP` parameters from user options ONLY if the `KSP` was previously set from options 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPResetFromOptions(KSP ksp)
```
Collective


## Input Parameter

- ***ksp -*** the `KSP` context
