# KSPGuessView
View the `KSPGuess` object 
## Synopsis
```
#include "petscksp.h"  
PetscErrorCode KSPGuessView(KSPGuess guess, PetscViewer view)
```
Logically Collective


## Input Parameters

- ***guess  -*** the initial guess object for the Krylov method
- ***viewer -*** the viewer object
