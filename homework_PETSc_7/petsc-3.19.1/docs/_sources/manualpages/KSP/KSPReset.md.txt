# KSPReset
Resets a `KSP` context to the kspsetupcalled = 0 state and removes any allocated Vecs and Mats 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPReset(KSP ksp)
```
Collective


## Input Parameter

- ***ksp -*** iterative context obtained from `KSPCreate()`
