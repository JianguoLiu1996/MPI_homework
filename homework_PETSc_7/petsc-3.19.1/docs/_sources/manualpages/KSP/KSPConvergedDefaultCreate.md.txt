# KSPConvergedDefaultCreate
Creates and initializes the context used by the `KSPConvergedDefault()` function 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPConvergedDefaultCreate(void **ctx)
```
Not Collective


## Output Parameter

- ***ctx -*** convergence context
