# MatLMVMClearJ0
Removes all definitions of J0 and reverts to an identity matrix (scale = 1.0). 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode MatLMVMClearJ0(Mat B)
```

## Input Parameter

- ***B -*** An LMVM-type matrix
