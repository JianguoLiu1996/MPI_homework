# KSPDestroy
Destroys `KSP` context. 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPDestroy(KSP *ksp)
```
Collective


## Input Parameter

- ***ksp -*** iterative context obtained from `KSPCreate()`
