# KSPSetUp
Sets up the internal data structures for the later use of an iterative solver. 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPSetUp(KSP ksp)
```
Collective


## Input Parameter

- ***ksp   -*** iterative context obtained from `KSPCreate()`

 [](chapter_ksp), `KSPCreate()`, `KSPSolve()`, `KSPDestroy()`, `KSP`
