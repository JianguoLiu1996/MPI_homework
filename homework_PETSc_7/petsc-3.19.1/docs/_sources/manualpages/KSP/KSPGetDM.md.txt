# KSPGetDM
Gets the `DM` that may be used by some preconditioners 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPGetDM(KSP ksp, DM *dm)
```
Not Collective


## Input Parameter

- ***ksp -*** the `KSP`



## Output Parameter

- ***dm -*** the `DM`
