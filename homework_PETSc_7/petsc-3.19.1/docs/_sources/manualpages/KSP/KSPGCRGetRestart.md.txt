# KSPGCRGetRestart
Gets number of iterations at which `KSPGCR` restarts. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPGCRGetRestart(KSP ksp, PetscInt *restart)
```
Not Collective


## Input Parameter

- ***ksp -*** the Krylov space context



## Output Parameter

- ***restart -*** integer restart value
