# KSPConvergedReasonViewCancel
Clears all the reasonview functions for a `KSP` object. 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPConvergedReasonViewCancel(KSP ksp)
```
Collective


## Input Parameter

- ***ksp -*** iterative context obtained from `KSPCreate()`
