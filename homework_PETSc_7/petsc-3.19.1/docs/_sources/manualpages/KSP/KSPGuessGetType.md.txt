# KSPGuessGetType
Gets the `KSPGuessType` as a string from the `KSPGuess` object. 
## Synopsis
```
#include "petscksp.h"  
PetscErrorCode KSPGuessGetType(KSPGuess guess, KSPGuessType *type)
```
Not Collective


## Input Parameter

- ***guess -*** the initial guess context



## Output Parameter

- ***name -*** type of `KSPGuess` method
