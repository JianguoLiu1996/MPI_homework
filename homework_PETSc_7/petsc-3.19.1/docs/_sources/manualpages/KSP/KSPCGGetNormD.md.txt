# KSPCGGetNormD
Got norm of the direction when the solver is used inside `SNESNEWTONTR` 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPCGGetNormD(KSP ksp, PetscReal *norm_d)
```
Collective


## Input Parameters

- ***ksp    -*** the iterative context
- ***norm_d -*** the norm of the direction
