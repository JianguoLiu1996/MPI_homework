# KSPGetType
Gets the `KSP` type as a string from the KSP object. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPGetType(KSP ksp, KSPType *type)
```
Not Collective


## Input Parameter

- ***ksp -*** Krylov context



## Output Parameter

- ***name -*** name of the `KSP` method
