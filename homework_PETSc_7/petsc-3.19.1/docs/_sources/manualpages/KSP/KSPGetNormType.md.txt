# KSPGetNormType
Gets the norm that is used for convergence testing. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPGetNormType(KSP ksp, KSPNormType *normtype)
```
Not Collective


## Input Parameter

- ***ksp -*** Krylov solver context



## Output Parameter

- ***normtype -*** norm that is used for convergence testing
