# MatLMVMGetJ0
Returns a pointer to the internal J0 matrix. 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode MatLMVMGetJ0(Mat B, Mat *J0)
```

## Input Parameters

- ***B -*** An LMVM-type matrix



## Output Parameter

- ***J0 -*** `Mat` object for defining the initial Jacobian
