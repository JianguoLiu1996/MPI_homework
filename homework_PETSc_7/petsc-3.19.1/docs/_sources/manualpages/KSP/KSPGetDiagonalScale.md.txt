# KSPGetDiagonalScale
Checks if `KSP` solver scales the matrix and right hand side, that is if `KSPSetDiagonalScale()` has been called 
## Synopsis
```
#include "petscksp.h" 
#include "petscmat.h" 
PetscErrorCode KSPGetDiagonalScale(KSP ksp, PetscBool *scale)
```
Not Collective


## Input Parameter

- ***ksp -*** the `KSP` context



## Output Parameter

- ***scale -*** `PETSC_TRUE` or `PETSC_FALSE`
