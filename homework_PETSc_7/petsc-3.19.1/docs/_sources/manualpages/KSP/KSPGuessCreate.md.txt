# KSPGuessCreate
Creates the default `KSPGuess` context. 
## Synopsis
```
#include "petscksp.h"  
PetscErrorCode KSPGuessCreate(MPI_Comm comm, KSPGuess *guess)
```
Collective


## Input Parameter

- ***comm -*** MPI communicator



## Output Parameter

- ***guess -*** location to put the `KSPGuess` context
