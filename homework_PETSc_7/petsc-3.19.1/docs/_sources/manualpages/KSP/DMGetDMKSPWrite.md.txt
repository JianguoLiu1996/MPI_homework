# DMGetDMKSPWrite
get write access to private `DMKSP` context from a `DM` 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode DMGetDMKSPWrite(DM dm, DMKSP *kspdm)
```
Logically Collective


## Input Parameter

- ***dm -*** `DM` used with a `KSP`



## Output Parameter

- ***kspdm -*** private `DMKSP` context
