# DMKSPCopy
copies the information in a `DMKSP` to another `DMKSP` 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode DMKSPCopy(DMKSP kdm, DMKSP nkdm)
```
Not Collective


## Input Parameters

- ***kdm -*** Original `DMKSP`
- ***nkdm -*** `DMKSP` to receive the data, created with `DMKSPCreate()`
