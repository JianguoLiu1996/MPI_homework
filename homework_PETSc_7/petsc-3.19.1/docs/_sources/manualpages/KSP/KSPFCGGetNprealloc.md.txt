# KSPFCGGetNprealloc
get the number of directions preallocate by `KSPFCG` 
## Synopsis
```
#include "petscksp.h"  
PetscErrorCode KSPFCGGetNprealloc(KSP ksp, PetscInt *nprealloc)
```
Not Collective


## Input Parameter

- ***ksp -*** the Krylov space context



## Output Parameter

- ***nprealloc -*** the number of directions preallocated
