# KSPCGGetObjFcn
Get objective function value when the solver is used inside `SNESNEWTONTR` 
## Synopsis
```
#include "petscksp.h" 
PetscErrorCode KSPCGGetObjFcn(KSP ksp, PetscReal *o_fcn)
```
Collective


## Input Parameters

- ***ksp   -*** the iterative context
- ***o_fcn -*** the objective function value
