# KSPFETIDPGetInnerKSP
Gets the `KSP` object for the Lagrange multipliers from inside a `KSPFETIDP` 
## Synopsis
```
#include <petscksp.h> 
PetscErrorCode KSPFETIDPGetInnerKSP(KSP ksp, KSP *innerksp)
```

## Input Parameters

- ***ksp -*** the `KSPFETIDP`
- ***innerksp -*** the `KSP` for the multipliers
