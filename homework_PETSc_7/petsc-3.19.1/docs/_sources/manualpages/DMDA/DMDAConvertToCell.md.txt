# DMDAConvertToCell
Convert (i,j,k) to local cell number 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDAConvertToCell(DM dm, MatStencil s, PetscInt *cell)
```
Not Collective


## Input Parameters

- ***da -*** the distributed array
- ***s -*** A `MatStencil` giving (i,j,k)



## Output Parameter

- ***cell -*** the local cell number
