# DMDAGetElementType
Gets the element type to be returned by `DMDAGetElements()` 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDAGetElementType(DM da, DMDAElementType *etype)
```
Not Collective


## Input Parameter

- ***da -*** the `DMDA` object



## Output Parameter

- ***etype -*** the element type, currently either `DMDA_ELEMENT_P1` or `DMDA_ELEMENT_Q1`
