# DMDARestoreSubdomainCornersIS
Restores the `IS` obtained with `DMDAGetSubdomainCornersIS()` 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDARestoreSubdomainCornersIS(DM dm, IS *is)
```
Not Collective


## Input Parameters

- ***dm -*** the `DM` object
- ***is -*** the index set
