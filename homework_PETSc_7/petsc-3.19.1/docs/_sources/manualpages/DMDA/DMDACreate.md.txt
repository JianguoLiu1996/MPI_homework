# DMDACreate
Creates a DMDA object. 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDACreate(MPI_Comm comm, DM *da)
```
Collective


## Input Parameter

- ***comm -*** The communicator for the DMDA object



## Output Parameter

- ***da  -*** The DMDA object





## Developers Note
Since there exists DMDACreate1/2/3d() should this routine even exist?
