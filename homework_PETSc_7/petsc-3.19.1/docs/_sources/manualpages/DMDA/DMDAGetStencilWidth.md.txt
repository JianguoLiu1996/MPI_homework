# DMDAGetStencilWidth
Gets the width of the communication stencil 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDAGetStencilWidth(DM da, PetscInt *width)
```
Not Collective


## Input Parameter

- ***da    -*** The `DMDA`



## Output Parameter

- ***width -*** The stencil width
