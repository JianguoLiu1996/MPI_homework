# DMDASetDof
Sets the number of degrees of freedom per vertex 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDASetDof(DM da, PetscInt dof)
```
Not Collective


## Input Parameters

- ***da  -*** The `DMDA`
- ***dof -*** Number of degrees of freedom
