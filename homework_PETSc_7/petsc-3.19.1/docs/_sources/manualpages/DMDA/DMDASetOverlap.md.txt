# DMDASetOverlap
Sets the size of the per-processor overlap. 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDASetOverlap(DM da, PetscInt x, PetscInt y, PetscInt z)
```
Not Collective


## Input Parameters

- ***da  -*** The `DMDA`
- ***x   -*** Overlap in the x direction
- ***y   -*** Overlap in the y direction
- ***z   -*** Overlap in the z direction
