# DMDAStencilType
Determines if the stencil extends only along the coordinate directions, or also to the northeast, northwest etc 
## Synopsis
```
typedef enum {
  DMDA_STENCIL_STAR,
  DMDA_STENCIL_BOX
} DMDAStencilType;
```



## See Also
 `DMDA`,  `DMDACreate1d()`, `DMDACreate2d()`, `DMDACreate3d()`, `DMDACreate()`, `DMDASetStencilType()`
