# DMDASetStencilType
Sets the type of the communication stencil 
## Synopsis
```
#include "petscdmda.h"   
PetscErrorCode DMDASetStencilType(DM da, DMDAStencilType stype)
```
Logically Collective


## Input Parameters

- ***da    -*** The `DMDA`
- ***stype -*** The stencil type, use either `DMDA_STENCIL_BOX` or `DMDA_STENCIL_STAR`.
