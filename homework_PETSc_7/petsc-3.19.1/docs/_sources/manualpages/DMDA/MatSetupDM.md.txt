# MatSetupDM
Sets the `DMDA` that is to be used by the HYPRE_StructMatrix PETSc matrix 
## Synopsis
```
#include "petscdmda.h"     
PetscErrorCode MatSetupDM(Mat mat, DM da)
```
Logically Collective


## Input Parameters

- ***mat -*** the matrix
- ***da -*** the da





## See Also
 `DMDA`, `Mat`, `MatSetUp()`
