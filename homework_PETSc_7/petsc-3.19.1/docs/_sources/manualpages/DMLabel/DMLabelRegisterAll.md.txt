# DMLabelRegisterAll
Registers all of the `DMLabel` implementations in the `DM` package. 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelRegisterAll(void)
```
Not Collective
