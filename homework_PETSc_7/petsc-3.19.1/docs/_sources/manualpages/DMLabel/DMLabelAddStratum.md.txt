# DMLabelAddStratum
Adds a new stratum value in a `DMLabel` 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelAddStratum(DMLabel label, PetscInt value)
```

## Input Parameters

- ***label -*** The `DMLabel`
- ***value -*** The stratum value
