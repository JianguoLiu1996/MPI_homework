# DMLabelClearValue
Clear the value a label assigns to a point 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelClearValue(DMLabel label, PetscInt point, PetscInt value)
```
Not Collective


## Input Parameters

- ***label -*** the `DMLabel`
- ***point -*** the point
- ***value -*** The point value
