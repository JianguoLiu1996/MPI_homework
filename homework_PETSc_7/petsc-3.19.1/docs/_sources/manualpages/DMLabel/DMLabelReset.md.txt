# DMLabelReset
Destroys internal data structures in a `DMLabel` 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelReset(DMLabel label)
```
Not Collective


## Input Parameter

- ***label -*** The `DMLabel`
