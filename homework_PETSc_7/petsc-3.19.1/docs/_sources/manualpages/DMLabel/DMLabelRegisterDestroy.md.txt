# DMLabelRegisterDestroy
This function destroys the `DMLabel` registry. It is called from `PetscFinalize()`. 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelRegisterDestroy(void)
```
