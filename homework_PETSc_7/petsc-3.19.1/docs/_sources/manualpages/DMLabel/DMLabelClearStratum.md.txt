# DMLabelClearStratum
Remove a stratum 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelClearStratum(DMLabel label, PetscInt value)
```
Not Collective


## Input Parameters

- ***label -*** the `DMLabel`
- ***value -*** the stratum value
