# ISBlockGetSize
Returns the global number of blocks in parallel in the index set of `ISType` `ISBLOCK` 
## Synopsis
```
#include "petscis.h"     
PetscErrorCode ISBlockGetSize(IS is, PetscInt *size)
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***size -*** the global number of blocks
