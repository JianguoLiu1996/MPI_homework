# ISShift
Shift all indices by given offset 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISShift(IS is, PetscInt offset, IS isy)
```
Collective


## Input Parameters

- ***is -*** the index set
- ***offset -*** the offset



## Output Parameter

- ***isy -*** the shifted copy of the input index set





## Notes
The `offset` can be different across processes.

`is` and `isy` can be the same.


## See Also
 `ISDuplicate()`, `ISCopy()`
