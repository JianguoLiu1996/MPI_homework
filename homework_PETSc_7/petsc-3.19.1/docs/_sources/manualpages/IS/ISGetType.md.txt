# ISGetType
Gets the index set type name, `ISType`, (as a string) from the `IS`. 
## Synopsis
```
#include "petscis.h"  
PetscErrorCode ISGetType(IS is, ISType *type)
```
Not Collective


## Input Parameter

- ***is  -*** The index set



## Output Parameter

- ***type -*** The index set type name
