# ISLocalToGlobalMappingSetBlockSize
Sets the blocksize of the mapping 
## Synopsis
```
#include "petscis.h"  
PetscErrorCode ISLocalToGlobalMappingSetBlockSize(ISLocalToGlobalMapping mapping, PetscInt bs)
```
Not Collective


## Input Parameters

- ***mapping -*** mapping data structure
- ***bs -*** the blocksize
