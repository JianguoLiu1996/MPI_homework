# ISGetLayout
get `PetscLayout` describing index set layout 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISGetLayout(IS is, PetscLayout *map)
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***map -*** the layout
