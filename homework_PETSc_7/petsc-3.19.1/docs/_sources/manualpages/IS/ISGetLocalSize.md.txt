# ISGetLocalSize
Returns the local (processor) length of an index set. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISGetLocalSize(IS is, PetscInt *size)
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***size -*** the local size





## See Also
 `IS`, `ISGetSize()`

beginner
