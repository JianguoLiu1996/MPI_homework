# ISDuplicate
Creates a duplicate copy of an index set. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISDuplicate(IS is, IS *newIS)
```
Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***isnew -*** the copy of the index set





## See Also
 `IS`, `ISCreateGeneral()`, `ISCopy()`

beginner
