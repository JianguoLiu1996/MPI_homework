# ISColoringView
Views an `ISColoring` coloring context. 
## Synopsis
```
#include "petscis.h"  
PetscErrorCode ISColoringView(ISColoring iscoloring, PetscViewer viewer)
```
Collective


## Input Parameters

- ***iscoloring -*** the coloring context
- ***viewer -*** the viewer
