# PetscLayoutSetLocalSize
Sets the local size for a `PetscLayout` object. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode PetscLayoutSetLocalSize(PetscLayout map, PetscInt n)
```
Collective


## Input Parameters

- ***map -*** pointer to the map
- ***n -*** the local size
