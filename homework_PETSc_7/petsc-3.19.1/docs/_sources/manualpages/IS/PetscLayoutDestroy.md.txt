# PetscLayoutDestroy
Frees a `PetscLayout` object and frees its range if that exists. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode PetscLayoutDestroy(PetscLayout *map)
```
Collective


## Input Parameter

- ***map -*** the `PetscLayout`
