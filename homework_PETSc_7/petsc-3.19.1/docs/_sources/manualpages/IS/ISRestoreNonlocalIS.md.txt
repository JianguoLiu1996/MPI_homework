# ISRestoreNonlocalIS
Restore the `IS` obtained with `ISGetNonlocalIS()`. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISRestoreNonlocalIS(IS is, IS *complement)
```
Not collective.


## Input Parameters

- ***is         -*** the index set
- ***complement -*** index set of `is`'s nonlocal indices
