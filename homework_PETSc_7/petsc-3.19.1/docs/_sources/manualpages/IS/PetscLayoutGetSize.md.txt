# PetscLayoutGetSize
Gets the global size for a `PetscLayout` object. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode PetscLayoutGetSize(PetscLayout map, PetscInt *n)
```
Not Collective


## Input Parameter

- ***map -*** pointer to the map



## Output Parameter

- ***n -*** the global size





## Note
Call this after the call to `PetscLayoutSetUp()`
