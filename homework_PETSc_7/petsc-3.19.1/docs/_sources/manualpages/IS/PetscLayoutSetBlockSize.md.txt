# PetscLayoutSetBlockSize
Sets the block size for a `PetscLayout` object. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode PetscLayoutSetBlockSize(PetscLayout map, PetscInt bs)
```
Logically Collective


## Input Parameters

- ***map -*** pointer to the map
- ***bs -*** the size
