# ISBlockGetLocalSize
Returns the local number of blocks in the index set of `ISType` `ISBLOCK` 
## Synopsis
```
#include "petscis.h"     
PetscErrorCode ISBlockGetLocalSize(IS is, PetscInt *size)
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***size -*** the local number of blocks
