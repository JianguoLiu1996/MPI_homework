# ISBlockRestoreIndices
Restores the indices associated with each block  in an `ISBLOCK` obtained with `ISBlockGetIndices()` 
## Synopsis
```
#include "petscis.h"     
PetscErrorCode ISBlockRestoreIndices(IS is, const PetscInt *idx[])
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***idx -*** the integer indices
