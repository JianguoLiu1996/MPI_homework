# PetscLayoutSetSize
Sets the global size for a `PetscLayout` object. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode PetscLayoutSetSize(PetscLayout map, PetscInt n)
```
Logically Collective


## Input Parameters

- ***map -*** pointer to the map
- ***n -*** the global size
