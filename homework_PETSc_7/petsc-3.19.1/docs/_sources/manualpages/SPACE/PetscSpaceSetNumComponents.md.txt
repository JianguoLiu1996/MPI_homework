# PetscSpaceSetNumComponents
Set the number of components for this space 
## Synopsis
```
#include "petscfe.h"  
PetscErrorCode PetscSpaceSetNumComponents(PetscSpace sp, PetscInt Nc)
```

## Input Parameters

- ***sp -*** The `PetscSpace`
- ***order -*** The number of components
