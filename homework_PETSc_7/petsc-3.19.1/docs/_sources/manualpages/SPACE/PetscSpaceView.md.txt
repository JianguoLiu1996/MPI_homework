# PetscSpaceView
Views a `PetscSpace` 
## Synopsis
```
#include "petscfe.h"  
PetscErrorCode PetscSpaceView(PetscSpace sp, PetscViewer v)
```
Collective


## Input Parameters

- ***sp -*** the `PetscSpace` object to view
- ***v  -*** the viewer
