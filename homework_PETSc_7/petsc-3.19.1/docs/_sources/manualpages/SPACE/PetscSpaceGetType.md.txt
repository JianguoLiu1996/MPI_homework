# PetscSpaceGetType
Gets the `PetscSpaceType` (as a string) from the object. 
## Synopsis
```
#include "petscfe.h"  
PetscErrorCode PetscSpaceGetType(PetscSpace sp, PetscSpaceType *name)
```
Not Collective


## Input Parameter

- ***sp  -*** The `PetscSpace`



## Output Parameter

- ***name -*** The `PetscSpace` type name
