# PetscSpaceGetNumVariables
Return the number of variables for this space 
## Synopsis
```
#include "petscfe.h"  
PetscErrorCode PetscSpaceGetNumVariables(PetscSpace sp, PetscInt *n)
```

## Input Parameter

- ***sp -*** The `PetscSpace`



## Output Parameter

- ***Nc -*** The number of variables, e.g. x, y, z...
