# MatGetNearNullSpace
Get null space attached with `MatSetNearNullSpace()` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatGetNearNullSpace(Mat mat, MatNullSpace *nullsp)
```
Not Collective


## Input Parameter

- ***mat -*** the matrix



## Output Parameter

- ***nullsp -*** the null space object, `NULL` if not set
