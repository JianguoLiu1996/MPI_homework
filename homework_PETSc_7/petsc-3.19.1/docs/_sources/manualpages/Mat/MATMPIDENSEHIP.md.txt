# MATMPIDENSEHIP
"mpidensehip" - A matrix type to be used for distributed dense matrices on GPUs. 
## Options Database Key

- ***-mat_type mpidensehip -*** sets the matrix type to `MATMPIDENSEHIP` during a call to
`MatSetFromOptions()`
