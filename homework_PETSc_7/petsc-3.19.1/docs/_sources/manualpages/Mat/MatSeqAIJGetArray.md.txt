# MatSeqAIJGetArray
gives read/write access to the array where the data for a `MATSEQAIJ` matrix is stored 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatSeqAIJGetArray(Mat A, PetscScalar **array)
```
Not Collective


## Input Parameter

- ***mat -*** a `MATSEQAIJ` matrix



## Output Parameter

- ***array -*** pointer to the data





## Fortran Note
`MatSeqAIJGetArray()` Fortran binding is deprecated (since PETSc 3.19), use `MatSeqAIJGetArrayF90()`
