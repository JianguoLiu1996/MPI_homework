# MATSEQDENSEHIP
"seqdensehip" - A matrix type to be used for sequential dense matrices on GPUs. 
## Options Database Keys

- ***-mat_type seqdensehip -*** sets the matrix type to `MATSEQDENSEHIP` during a call to
`MatSetFromOptions()`
