# MatNorm
Calculates various norms of a matrix. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatNorm(Mat mat, NormType type, PetscReal *nrm)
```
Collective


## Input Parameters

- ***mat -*** the matrix
- ***type -*** the type of norm, `NORM_1`, `NORM_FROBENIUS`, `NORM_INFINITY`



## Output Parameter

- ***nrm -*** the resulting norm
