# MatH2OpusCompress
Compress a hierarchical matrix. 
## Synopsis
```
PetscErrorCode MatH2OpusCompress(Mat A, PetscReal tol)
```

## Input Parameters

- ***A -*** the matrix
- ***tol -*** the absolute truncation threshold
