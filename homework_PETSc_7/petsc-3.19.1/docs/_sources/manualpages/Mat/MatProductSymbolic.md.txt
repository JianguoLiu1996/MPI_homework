# MatProductSymbolic
Perform the symbolic portion of a matrix-matrix product operation, this creates a data structure for use with the numerical product to be done with `MatProductNumeric()` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatProductSymbolic(Mat mat)
```
Collective


## Input/Output Parameter

- ***mat -*** the matrix whose values are to be computed via a matrix-matrix product operation





## Note
`MatProductSetFromOptions()` must have been called on `mat` before calling this function
