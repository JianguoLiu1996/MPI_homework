# MatNullSpaceView
Visualizes a null space object. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatNullSpaceView(MatNullSpace sp, PetscViewer viewer)
```
Collective; No Fortran Support


## Input Parameters

- ***matnull -*** the null space
- ***viewer -*** visualization context
