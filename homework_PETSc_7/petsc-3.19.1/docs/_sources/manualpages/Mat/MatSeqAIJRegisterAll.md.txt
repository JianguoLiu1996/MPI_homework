# MatSeqAIJRegisterAll
Registers all of the matrix subtypes of `MATSSEQAIJ` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatSeqAIJRegisterAll(void)
```
Not Collective




## Note
This registers the versions of `MATSEQAIJ` for GPUs
