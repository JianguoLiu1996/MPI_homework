# MatGetDiagonal
Gets the diagonal of a matrix as a `Vec` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatGetDiagonal(Mat mat, Vec v)
```
Logically Collective


## Input Parameter

- ***mat -*** the matrix



## Output Parameter

- ***v -*** the diagonal of the matrix





## Note
Currently only correct in parallel for square matrices.
