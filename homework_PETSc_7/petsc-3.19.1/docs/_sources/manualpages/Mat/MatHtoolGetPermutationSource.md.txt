# MatHtoolGetPermutationSource
Gets the permutation associated to the source cluster for a `MATHTOOL` matrix. 
## Synopsis
```
#include "petscmat.h" 
PETSC_EXTERN PetscErrorCode MatHtoolGetPermutationSource(Mat A, IS *is)
```

## Input Parameter

- ***A -*** hierarchical matrix



## Output Parameter

- ***is -*** permutation
