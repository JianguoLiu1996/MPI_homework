# MATMPIDENSECUDA
"mpidensecuda" - A matrix type to be used for distributed dense matrices on GPUs. 
## Options Database Key

- ***-mat_type mpidensecuda -*** sets the matrix type to `MATMPIDENSECUDA` during a call to
`MatSetFromOptions()`
