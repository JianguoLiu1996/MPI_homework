# MatMult
Computes the matrix-vector product, y = Ax. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatMult(Mat mat, Vec x, Vec y)
```
Neighbor-wise Collective


## Input Parameters

- ***mat -*** the matrix
- ***x   -*** the vector to be multiplied



## Output Parameter

- ***y -*** the result





## Note
The vectors `x` and `y` cannot be the same.  I.e., one cannot
call `MatMult`(A,y,y).
