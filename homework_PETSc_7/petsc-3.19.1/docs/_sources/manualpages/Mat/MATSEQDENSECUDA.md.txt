# MATSEQDENSECUDA
"seqdensecuda" - A matrix type to be used for sequential dense matrices on GPUs. 
## Options Database Keys

- ***-mat_type seqdensecuda -*** sets the matrix type to `MATSEQDENSECUDA` during a call to
`MatSetFromOptions()`
