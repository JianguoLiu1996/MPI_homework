# MatGetTrace
Gets the trace of a matrix. The sum of the diagonal entries. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatGetTrace(Mat mat, PetscScalar *trace)
```
Collective


## Input Parameter

- ***mat -*** the matrix



## Output Parameter

- ***trace -*** the sum of the diagonal entries
