# MatISSetLocalMatType
Specifies the type of local matrix inside the `MATIS` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatISSetLocalMatType(Mat mat, MatType mtype)
```

## Input Parameters

- ***mat -*** the matrix
- ***mtype -*** the local matrix type
