# MATMPISELL
MATMPISELL = "MPISELL" - A matrix type to be used for parallel sparse matrices. 
## Options Database Keys

- ***-mat_type mpisell -*** sets the matrix type to `MATMPISELL` during a call to `MatSetFromOptions()`
