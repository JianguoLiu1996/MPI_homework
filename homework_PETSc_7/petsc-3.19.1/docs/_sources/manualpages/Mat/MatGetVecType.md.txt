# MatGetVecType
Gets the vector type the matrix will return with `MatCreateVecs()` 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatGetVecType(Mat mat, VecType *vtype)
```
Not Collective


## Input Parameter

- ***mat -*** the matrix



## Output Parameter

- ***name -*** name of vector type
