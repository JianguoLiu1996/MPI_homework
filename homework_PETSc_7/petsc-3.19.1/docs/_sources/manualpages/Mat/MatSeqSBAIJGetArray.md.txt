# MatSeqSBAIJGetArray
gives access to the array where the numerical data for a `MATSEQSBAIJ` matrix is stored 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatSeqSBAIJGetArray(Mat A, PetscScalar **array)
```
Not Collective


## Input Parameter

- ***mat -*** a `MATSEQSBAIJ` matrix



## Output Parameter

- ***array -*** pointer to the data
