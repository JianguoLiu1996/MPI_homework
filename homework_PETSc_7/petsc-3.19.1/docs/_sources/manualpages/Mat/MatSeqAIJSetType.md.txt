# MatSeqAIJSetType
Converts a `MATSEQAIJ` matrix to a subtype 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatSeqAIJSetType(Mat mat, MatType matype)
```
Collective


## Input Parameters

- ***mat      -*** the matrix object
- ***matype   -*** matrix type



## Options Database Key

- ***-mat_seqaij_type  <method> -*** for example seqaijcrl
