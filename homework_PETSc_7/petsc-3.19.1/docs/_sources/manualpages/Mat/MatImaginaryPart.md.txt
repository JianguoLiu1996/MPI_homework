# MatImaginaryPart
Moves the imaginary part of the matrix to the real part and zeros the imaginary part 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatImaginaryPart(Mat mat)
```
Logically Collective


## Input Parameter

- ***mat -*** the matrix
