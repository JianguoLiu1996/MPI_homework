# MatGetLayouts
Gets the `PetscLayout` objects for rows and columns 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatGetLayouts(Mat A, PetscLayout *rmap, PetscLayout *cmap)
```
Not Collective


## Input Parameter

- ***A -*** the matrix



## Output Parameters

- ***rmap -*** row layout
- ***cmap -*** column layout
