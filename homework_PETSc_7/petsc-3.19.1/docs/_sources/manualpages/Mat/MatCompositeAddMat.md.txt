# MatCompositeAddMat
Add another matrix to a composite matrix. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatCompositeAddMat(Mat mat, Mat smat)
```
Collective


## Input Parameters

- ***mat -*** the composite matrix
- ***smat -*** the partial matrix
