# MatTransposeColoringDestroy
Destroys a coloring context for matrix product C=A*B^T that was created via `MatTransposeColoringCreate()`. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatTransposeColoringDestroy(MatTransposeColoring *c)
```
Collective


## Input Parameter

- ***c -*** coloring context
