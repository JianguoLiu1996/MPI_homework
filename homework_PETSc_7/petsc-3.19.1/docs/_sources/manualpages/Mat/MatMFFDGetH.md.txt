# MatMFFDGetH
Gets the last value that was used as the differencing for a `MATMFFD` matrix parameter. 
## Synopsis
```
#include "petscmat.h"   
PetscErrorCode MatMFFDGetH(Mat mat, PetscScalar *h)
```
Not Collective


## Input Parameters

- ***mat -*** the `MATMFFD` matrix



## Output Parameter

- ***h -*** the differencing step size
