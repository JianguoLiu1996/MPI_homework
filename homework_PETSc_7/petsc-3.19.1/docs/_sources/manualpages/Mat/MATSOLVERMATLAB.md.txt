# MATSOLVERMATLAB
"matlab" - Providing direct solver LU for `MATSEQAIJ` matrix via the external package MATLAB. Use `./configure` with the options `--with-matlab` to install PETSc with this capability


## Options Database Key

- ***-pc_factor_mat_solver_type matlab -*** selects MATLAB to do the sparse factorization
