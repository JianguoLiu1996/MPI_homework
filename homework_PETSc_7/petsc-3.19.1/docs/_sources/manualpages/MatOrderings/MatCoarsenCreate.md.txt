# MatCoarsenCreate
Creates a coarsen context. 
## Synopsis
```
#include "petscmatcoarsen.h" 
PetscErrorCode MatCoarsenCreate(MPI_Comm comm, MatCoarsen *newcrs)
```
Collective


## Input Parameter

- ***comm -*** MPI communicator



## Output Parameter

- ***newcrs -*** location to put the context
