# MatPartitioningChacoGetEigenTol
Gets the eigensolver tolerance used by Chaco 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatPartitioningChacoGetEigenTol(MatPartitioning part, PetscReal *tol)
```
Not Collective


## Input Parameter

- ***part -*** the partitioning context



## Output Parameter

- ***tol  -*** the tolerance
