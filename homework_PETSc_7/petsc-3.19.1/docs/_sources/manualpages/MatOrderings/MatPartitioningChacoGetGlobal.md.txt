# MatPartitioningChacoGetGlobal
Get the global method used by the Chaco partitioner. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatPartitioningChacoGetGlobal(MatPartitioning part, MPChacoGlobalType *method)
```
Not Collective


## Input Parameter

- ***part -*** the partitioning context



## Output Parameter

- ***method -*** the method
