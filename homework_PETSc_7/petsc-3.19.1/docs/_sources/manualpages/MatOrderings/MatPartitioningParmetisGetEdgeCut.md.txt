# MatPartitioningParmetisGetEdgeCut
Returns the number of edge cuts in the vertex partition. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatPartitioningParmetisGetEdgeCut(MatPartitioning part, PetscInt *cut)
```

## Input Parameter

- ***part -*** the partitioning context



## Output Parameter

- ***cut -*** the edge cut
