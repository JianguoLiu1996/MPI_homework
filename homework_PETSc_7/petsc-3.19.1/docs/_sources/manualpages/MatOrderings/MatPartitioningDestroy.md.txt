# MatPartitioningDestroy
Destroys the partitioning context. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatPartitioningDestroy(MatPartitioning *part)
```
Collective


## Input Parameter

- ***part -*** the partitioning context
