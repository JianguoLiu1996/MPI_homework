# MatPartitioningCreate
Creates a partitioning context. 
## Synopsis
```
#include "petscmat.h" 
PetscErrorCode MatPartitioningCreate(MPI_Comm comm, MatPartitioning *newp)
```
Collective


## Input Parameter

- ***comm -*** MPI communicator



## Output Parameter

- ***newp -*** location to put the context
