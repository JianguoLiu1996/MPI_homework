# SNESGetType
Gets the `SNES` method type and name (as a string). 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESGetType(SNES snes, SNESType *type)
```
Not Collective


## Input Parameter

- ***snes -*** nonlinear solver context



## Output Parameter

- ***type -*** `SNES` method (a character string)
