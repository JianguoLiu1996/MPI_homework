# DMPlexSNESComputeBoundaryFEM
Form the boundary values for the local input X 
## Synopsis
```
#include "petscdmplex.h" 
#include "petscsnes.h"   
PetscErrorCode DMPlexSNESComputeBoundaryFEM(DM dm, Vec X, void *user)
```

## Input Parameters

- ***dm -*** The mesh
- ***user -*** The user context



## Output Parameter

- ***X  -*** Local solution
