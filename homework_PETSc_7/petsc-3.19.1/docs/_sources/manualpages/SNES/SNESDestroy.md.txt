# SNESDestroy
Destroys the nonlinear solver context that was created with `SNESCreate()`. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESDestroy(SNES *snes)
```
Collective


## Input Parameter

- ***snes -*** the `SNES` context
