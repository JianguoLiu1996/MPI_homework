# SNESAddOptionsChecker
Adds an additional function to check for `SNES` options. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESAddOptionsChecker(PetscErrorCode (*snescheck)(SNES))
```
Not Collective


## Input Parameter

- ***snescheck -*** function that checks for options
