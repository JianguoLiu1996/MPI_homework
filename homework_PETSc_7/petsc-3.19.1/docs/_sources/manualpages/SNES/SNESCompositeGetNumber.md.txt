# SNESCompositeGetNumber
Get the number of subsolvers in the `SNESCOMPOSITE` 
## Synopsis
```
#include "petscsnes.h" 
PetscErrorCode SNESCompositeGetNumber(SNES snes, PetscInt *n)
```
Logically Collective


## Input Parameter

- ***snes -*** the snes context



## Output Parameter

- ***n -*** the number of subsolvers
