# SNESNGSFunction
function used to apply a Gauss-Seidel sweep on the nonlinear function 
## Synopsis
```
#include <petscsnes.h>
$    SNESNGSFunction(SNES snes,Vec x,Vec b,void *ctx);
```
Collective


## Input Parameters

- ***X   -*** solution vector
- ***B   -*** RHS vector
- ***ctx -*** optional user-defined Gauss-Seidel context



## Output Parameter

- ***X   -*** solution vector
