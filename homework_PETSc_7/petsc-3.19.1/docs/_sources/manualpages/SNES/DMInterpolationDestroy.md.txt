# DMInterpolationDestroy
Destroys a `DMInterpolationInfo` context 
## Synopsis
```
#include "petscdmplex.h" 
#include "petscsnes.h"   
PetscErrorCode DMInterpolationDestroy(DMInterpolationInfo *ctx)
```
Collective


## Input Parameter

- ***ctx -*** the context
