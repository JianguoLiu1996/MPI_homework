# SNESMSGetType
Get the type of multistage smoother `SNESMS` 
## Synopsis
```
#include "petscsnes.h" 
PetscErrorCode SNESMSGetType(SNES snes, SNESMSType *mstype)
```
Not Collective


## Input Parameter

- ***snes -*** nonlinear solver context



## Output Parameter

- ***mstype -*** type of multistage method
