# MatSNESMFGetSNES
returns the `SNES` associated with a matrix created with `MatCreateSNESMF()` 
## Synopsis
```
#include "petscsnes.h" 
#include "petscdm.h"   
PetscErrorCode MatSNESMFGetSNES(Mat J, SNES *snes)
```
Not Collective


## Input Parameter

- ***J -*** the matrix



## Output Parameter

- ***snes -*** the `SNES` object





## See Also
 `Mat`, `SNES`, `MatCreateSNESMF()`
