# SNESSetIterationNumber
Sets the current iteration number. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESSetIterationNumber(SNES snes, PetscInt iter)
```
Not Collective


## Input Parameters

- ***snes -*** `SNES` context
- ***iter -*** iteration number
