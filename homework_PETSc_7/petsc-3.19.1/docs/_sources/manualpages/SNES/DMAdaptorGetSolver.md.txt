# DMAdaptorGetSolver
Gets the solver used to produce discrete solutions 
## Synopsis
```
#include "petscdmadaptor.h" 
PetscErrorCode DMAdaptorGetSolver(DMAdaptor adaptor, SNES *snes)
```
Not Collective


## Input Parameter

- ***adaptor   -*** The `DMAdaptor` object



## Output Parameter

- ***snes -*** The solver
