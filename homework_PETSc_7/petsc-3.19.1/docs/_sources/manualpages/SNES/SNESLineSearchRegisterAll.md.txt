# SNESLineSearchRegisterAll
Registers all of the nonlinear solver methods in the `SNESLineSearch` package. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESLineSearchRegisterAll(void)
```
Not Collective
