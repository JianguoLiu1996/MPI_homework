# SNESConvergedReasonViewCancel
Clears all the reason view functions for a `SNES` object. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESConvergedReasonViewCancel(SNES snes)
```
Collective


## Input Parameter

- ***snes -*** iterative context obtained from `SNESCreate()`
