# SNESSetSolution
Sets the solution vector for use by the `SNES` routines. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESSetSolution(SNES snes, Vec u)
```
Logically Collective


## Input Parameters

- ***snes -*** the `SNES` context obtained from `SNESCreate()`
- ***u    -*** the solution vector
