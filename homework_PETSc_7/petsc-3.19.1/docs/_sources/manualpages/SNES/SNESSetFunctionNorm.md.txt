# SNESSetFunctionNorm
Sets the last computed residual norm. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESSetFunctionNorm(SNES snes, PetscReal norm)
```
Logically Collective


## Input Parameters

- ***snes -*** the `SNES` context
- ***norm -*** the value of the norm
