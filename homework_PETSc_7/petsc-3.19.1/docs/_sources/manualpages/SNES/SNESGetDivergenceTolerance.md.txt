# SNESGetDivergenceTolerance
Gets divergence tolerance used in divergence test. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESGetDivergenceTolerance(SNES snes, PetscReal *divtol)
```
Not Collective


## Input Parameters

- ***snes -*** the `SNES` context
- ***divtol -*** divergence tolerance
