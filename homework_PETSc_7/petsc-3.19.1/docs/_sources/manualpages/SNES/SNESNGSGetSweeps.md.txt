# SNESNGSGetSweeps
Gets the number of sweeps nonlinear GS will use in `SNESNCG` 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESNGSGetSweeps(SNES snes, PetscInt *sweeps)
```

## Input Parameter

- ***snes   -*** the `SNES` context



## Output Parameter

- ***sweeps  -*** the number of sweeps of nonlinear GS to perform.
