# SNESHasNPC
Returns whether a nonlinear preconditioner exists 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESHasNPC(SNES snes, PetscBool *has_npc)
```
Not Collective


## Input Parameter

- ***snes -*** iterative context obtained from `SNESCreate()`



## Output Parameter

- ***has_npc -*** whether the `SNES` has an NPC or not
