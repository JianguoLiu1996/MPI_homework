# SNESNASMGetNumber
Gets number of subsolvers 
## Synopsis
```
#include "petscsnes.h"   
PetscErrorCode SNESNASMGetNumber(SNES snes, PetscInt *n)
```
Not Collective


## Input Parameter

- ***snes -*** the `SNES` context



## Output Parameter

- ***n -*** the number of subsolvers

 `SNESNASM`, `SNESNASMGetSNES()`
