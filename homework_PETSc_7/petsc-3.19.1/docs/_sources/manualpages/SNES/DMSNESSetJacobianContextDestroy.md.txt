# DMSNESSetJacobianContextDestroy
set `SNES` Jacobian evaluation context destroy function 
## Synopsis
```
#include "petscsnes.h" 
#include "petscdm.h" 
PetscErrorCode DMSNESSetJacobianContextDestroy(DM dm, PetscErrorCode (*f)(void *))
```
Not Collective


## Input Parameters

- ***dm -*** `DM` to be used with `SNES`
- ***f -*** Jacobian evaluation context destroy function
