# SNESGetLineSearch
Returns the line search context set with `SNESSetLineSearch()` or creates a default line search instance associated with the `SNES` and returns it. 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESGetLineSearch(SNES snes, SNESLineSearch *linesearch)
```
Not Collective


## Input Parameter

- ***snes -*** iterative context obtained from `SNESCreate()`



## Output Parameter

- ***linesearch -*** linesearch context
