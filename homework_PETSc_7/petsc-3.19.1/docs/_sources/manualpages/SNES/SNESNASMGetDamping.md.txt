# SNESNASMGetDamping
Gets the update damping for `SNESNASM` the nonlinear additive Schwarz solver 
## Synopsis
```
#include "petscsnes.h"   
PetscErrorCode SNESNASMGetDamping(SNES snes, PetscReal *dmp)
```
Not Collective


## Input Parameter

- ***snes -*** the SNES context



## Output Parameter

- ***dmp -*** damping





## See Also
 `SNESNASM`, `SNESNASMSetDamping()`
