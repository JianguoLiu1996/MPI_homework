# SNESCompositeAddSNES
Adds another `SNES` to the `SNESCOMPOSITE` 
## Synopsis
```
#include "petscsnes.h" 
PetscErrorCode SNESCompositeAddSNES(SNES snes, SNESType type)
```
Collective


## Input Parameters

- ***snes -*** the snes context of type `SNESCOMPOSITE`
- ***type -*** the type of the new solver
