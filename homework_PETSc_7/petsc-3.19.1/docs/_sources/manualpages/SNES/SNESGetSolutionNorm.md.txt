# SNESGetSolutionNorm
Gets the last computed norm of the solution 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESGetSolutionNorm(SNES snes, PetscReal *xnorm)
```
Not Collective


## Input Parameter

- ***snes -*** the `SNES` context



## Output Parameter

- ***xnorm -*** the last computed solution norm
