# PetscHSetIJGetCapacity
Get the current size of the array in the hash set 
## Synopsis
```
#include <petsc/private/hashsetij.h>
PetscErrorCode PetscHSetIJGetCapacity(PetscHSetIJ ht,PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash set



## Output Parameter

- ***n -*** The capacity
