# PetscHSetIQueryDel
Query and remove an entry from the hash set 
## Synopsis
```
#include <petsc/private/hashseti.h>
PetscErrorCode PetscHSetIQueryDel(PetscHSetI ht, PetscInt key, PetscBool *present)
```

## Input Parameters

- ***ht  -*** The hash set
- ***key -*** The entry



## Output Parameter

- ***present -*** Boolean indicating whether the entry was present

 `PetscHSetIQueryAdd()`, `PetscHSetIDel()`
