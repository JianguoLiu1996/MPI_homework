# PetscHSetIResize
Set the number of buckets in a hash set 
## Synopsis
```
#include <petsc/private/hashseti.h>
PetscErrorCode PetscHSetIResize(PetscHSetI ht,PetscInt nb)
```

## Input Parameters

- ***ht -*** The hash set
- ***nb -*** The number of buckets
