# PetscHMapIVIterSet
Set the value referenced by an iterator in the hash 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVIterSet(PetscHMapIV ht, PetscHashIter iter, PetscScalar val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
- ***val  -*** The value
