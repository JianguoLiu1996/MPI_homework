# PetscHMapIIterDel
Remove the (key,value) referenced by an iterator from the hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIIterDel(PetscHMapI ht, PetscHashIter iter)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
