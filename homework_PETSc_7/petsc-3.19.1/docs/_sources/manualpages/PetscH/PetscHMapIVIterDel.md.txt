# PetscHMapIVIterDel
Remove the (key,value) referenced by an iterator from the hash table 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVIterDel(PetscHMapIV ht, PetscHashIter iter)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
