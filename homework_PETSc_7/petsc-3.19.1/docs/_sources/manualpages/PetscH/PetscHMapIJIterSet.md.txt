# PetscHMapIJIterSet
Set the value referenced by an iterator in the hash 
## Synopsis
```
#include <petsc/private/hashmapij.h>
PetscErrorCode PetscHMapIJIterSet(PetscHMapIJ ht, PetscHashIter iter, PetscScalar val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
- ***val  -*** The value
