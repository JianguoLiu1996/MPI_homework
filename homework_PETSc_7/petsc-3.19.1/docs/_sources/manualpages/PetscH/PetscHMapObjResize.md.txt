# PetscHMapObjResize
Set the number of buckets in a hash table 
## Synopsis
```
#include <petsc/private/hashmapobj.h>
PetscErrorCode PetscHMapObjResize(PetscHMapObj ht, PetscInt nb)
```

## Input Parameters

- ***ht -*** The hash table
- ***nb -*** The number of buckets
