# PetscHMapIGetSize
Get the number of entries in a hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIGetSize(PetscHMapI ht, PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash table



## Output Parameter

- ***n -*** The number of entries
