# PetscHMapIHas
Query for a key in the hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIHas(PetscHMapI ht, PetscInt key, PetscBool *has)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key



## Output Parameter

- ***has -*** Boolean indicating whether key is in the hash table
