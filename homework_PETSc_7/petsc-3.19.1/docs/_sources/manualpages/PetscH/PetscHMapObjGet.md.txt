# PetscHMapObjGet
Get the value for a key in the hash table 
## Synopsis
```
#include <petsc/private/hashmapobj.h>
PetscErrorCode PetscHMapObjGet(PetscHMapObj ht, PetscInt64 key, PetscObject *val)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key



## Output Parameter

- ***val -*** The value
