# PetscHMapIGetCapacity
Get the current size of the array in the hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIGetCapacity(PetscHMapI ht, PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash table



## Output Parameter

- ***n -*** The capacity
