# PetscHMapIVDel
Remove a key and its value from the hash table 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVDel(PetscHMapIV ht,PetscInt key)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
