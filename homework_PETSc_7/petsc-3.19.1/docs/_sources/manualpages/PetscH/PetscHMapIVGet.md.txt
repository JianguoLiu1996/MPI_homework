# PetscHMapIVGet
Get the value for a key in the hash table 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVGet(PetscHMapIV ht, PetscInt key, PetscScalar *val)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key



## Output Parameter

- ***val -*** The value
