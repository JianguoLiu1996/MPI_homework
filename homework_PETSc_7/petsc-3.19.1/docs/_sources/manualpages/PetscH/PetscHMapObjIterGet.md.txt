# PetscHMapObjIterGet
Get the value referenced by an iterator in the hash table 
## Synopsis
```
#include <petsc/private/hashmapobj.h>
PetscErrorCode PetscHMapObjIterGet(PetscHMapObj ht, PetscHashIter iter, PetscObject *val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator



## Output Parameter

- ***val  -*** The value
