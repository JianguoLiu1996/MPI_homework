# PetscHMapIDel
Remove a key and its value from the hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIDel(PetscHMapI ht,PetscInt key)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
