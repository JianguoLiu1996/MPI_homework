# PetscHSetIGetSize
Get the number of entries in a hash set 
## Synopsis
```
#include <petsc/private/hashseti.h>
PetscErrorCode PetscHSetIGetSize(PetscHSetI ht,PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash set



## Output Parameter

- ***n -*** The number of entries
