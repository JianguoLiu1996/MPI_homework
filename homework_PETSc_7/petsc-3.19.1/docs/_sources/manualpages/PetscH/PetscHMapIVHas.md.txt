# PetscHMapIVHas
Query for a key in the hash table 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVHas(PetscHMapIV ht, PetscInt key, PetscBool *has)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key



## Output Parameter

- ***has -*** Boolean indicating whether key is in the hash table
