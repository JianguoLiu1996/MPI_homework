# PetscHMapIIterSet
Set the value referenced by an iterator in the hash 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIIterSet(PetscHMapI ht, PetscHashIter iter, PetscInt val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
- ***val  -*** The value
