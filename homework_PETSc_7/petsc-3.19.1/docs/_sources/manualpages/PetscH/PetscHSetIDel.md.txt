# PetscHSetIDel
Remove an entry from the hash set 
## Synopsis
```
#include <petsc/private/hashseti.h>
PetscErrorCode PetscHSetIDel(PetscHSetI ht, PetscInt key)
```

## Input Parameters

- ***ht  -*** The hash set
- ***key -*** The entry
