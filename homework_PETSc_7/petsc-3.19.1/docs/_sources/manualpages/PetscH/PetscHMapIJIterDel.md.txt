# PetscHMapIJIterDel
Remove the (key,value) referenced by an iterator from the hash table 
## Synopsis
```
#include <petsc/private/hashmapij.h>
PetscErrorCode PetscHMapIJIterDel(PetscHMapIJ ht, PetscHashIter iter)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator
