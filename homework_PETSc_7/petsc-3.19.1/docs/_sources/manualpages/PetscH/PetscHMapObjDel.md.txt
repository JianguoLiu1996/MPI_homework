# PetscHMapObjDel
Remove a key and its value from the hash table 
## Synopsis
```
#include <petsc/private/hashmapobj.h>
PetscErrorCode PetscHMapObjDel(PetscHMapObj ht,PetscInt64 key)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
