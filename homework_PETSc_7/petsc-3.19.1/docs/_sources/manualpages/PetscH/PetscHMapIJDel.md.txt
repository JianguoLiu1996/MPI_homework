# PetscHMapIJDel
Remove a key and its value from the hash table 
## Synopsis
```
#include <petsc/private/hashmapij.h>
PetscErrorCode PetscHMapIJDel(PetscHMapIJ ht,struct {PetscInt i, j;} key)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
