# PetscHSetIGetCapacity
Get the current size of the array in the hash set 
## Synopsis
```
#include <petsc/private/hashseti.h>
PetscErrorCode PetscHSetIGetCapacity(PetscHSetI ht,PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash set



## Output Parameter

- ***n -*** The capacity
