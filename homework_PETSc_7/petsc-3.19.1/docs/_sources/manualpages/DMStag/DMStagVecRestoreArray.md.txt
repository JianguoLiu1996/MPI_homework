# DMStagVecRestoreArray
restore access to a raw array 
## Synopsis
```
PetscErrorCode DMStagVecRestoreArray(DM dm, Vec vec, void *array)
```
Logically Collective


## Input Parameters

- ***dm -*** the `DMSTAG` object
- ***vec -*** the `Vec` object



## Output Parameter

- ***array -*** the array
