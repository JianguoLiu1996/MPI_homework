# DMStagStencilType
Elementwise stencil type, determining which neighbors participate in communication 
## Synopsis
```
typedef enum {
  DMSTAG_STENCIL_NONE = 0,
  DMSTAG_STENCIL_STAR,
  DMSTAG_STENCIL_BOX
} DMStagStencilType;
```
