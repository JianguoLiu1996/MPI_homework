# DMStagGetStencilWidth
get elementwise stencil width 
## Synopsis
```
PetscErrorCode DMStagGetStencilWidth(DM dm, PetscInt *stencilWidth)
```
Not Collective


## Input Parameter

- ***dm -*** the `DMSTAG` object



## Output Parameter

- ***stencilWidth -*** stencil/halo/ghost width in elements
