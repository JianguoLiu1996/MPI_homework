# DMStagVecRestoreArrayRead
restore read-only access to a raw array 
## Synopsis
```
PetscErrorCode DMStagVecRestoreArrayRead(DM dm, Vec vec, void *array)
```
Logically Collective


## Input Parameters

- ***dm -*** the `DMSTAG` object
- ***vec -*** the Vec object



## Output Parameter

- ***array -*** the read-only array
