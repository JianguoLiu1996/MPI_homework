# DMPlexGetMigrationSF
Gets the `PetscSF` for migrating from a parent `DM` into this `DM` 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetMigrationSF(DM dm, PetscSF *migrationSF)
```
Note Collective


## Input Parameter

- ***dm          -*** The `DM`



## Output Parameter

- ***migrationSF -*** The `PetscSF`
