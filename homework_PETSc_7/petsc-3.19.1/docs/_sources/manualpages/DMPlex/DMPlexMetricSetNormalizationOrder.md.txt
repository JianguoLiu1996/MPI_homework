# DMPlexMetricSetNormalizationOrder
Set the order p for L-p normalization 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexMetricSetNormalizationOrder(DM dm, PetscReal p)
```

## Input parameters

- ***dm -*** The `DM`
- ***p  -*** The normalization order
