# DMPlexTransformDestroy
Destroys a `DMPlexTransform` 
## Synopsis
```
#include "petscdmplextransform.h" 
PetscErrorCode DMPlexTransformDestroy(DMPlexTransform *tr)
```
Collective


## Input Parameter

- ***tr -*** the transform object to destroy
