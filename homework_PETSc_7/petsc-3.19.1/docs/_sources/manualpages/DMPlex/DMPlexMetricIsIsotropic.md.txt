```
#include "petscdmplex.h"   
PetscErrorCode DMPlexMetricIsIsotropic(DM dm, PetscBool *isotropic)
```

## Input parameters

- ***dm        -*** The `DM`



## Output parameters

- ***isotropic -*** Is the metric isotropic?
