# DMPlexGetCellType
Get the polytope type of a given cell 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetCellType(DM dm, PetscInt cell, DMPolytopeType *celltype)
```
Not Collective


## Input Parameters

- ***dm   -*** The `DMPLEX` object
- ***cell -*** The cell



## Output Parameter

- ***celltype -*** The polytope type of the cell
