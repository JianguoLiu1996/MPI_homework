# DMPlexComputeInjectorFEM
Compute a mapping from coarse unknowns to fine unknowns 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexComputeInjectorFEM(DM dmc, DM dmf, VecScatter *sc, void *user)
```

## Input Parameters

- ***dmc  -*** The coarse mesh
- ***dmf  -*** The fine mesh


- ***user -*** The user context



## Output Parameter

- ***sc   -*** The mapping
