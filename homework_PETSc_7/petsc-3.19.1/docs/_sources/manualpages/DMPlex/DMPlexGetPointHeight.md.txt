# DMPlexGetPointHeight
Get the `height` of a given point 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetPointHeight(DM dm, PetscInt point, PetscInt *height)
```
Not Collective


## Input Parameters

- ***dm    -*** The `DMPLEX` object
- ***point -*** The point



## Output Parameter

- ***height -*** The height of the `point`
