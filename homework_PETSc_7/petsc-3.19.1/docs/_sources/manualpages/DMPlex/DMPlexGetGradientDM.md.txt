# DMPlexGetGradientDM
Return gradient data layout 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetGradientDM(DM dm, PetscFV fv, DM *dmGrad)
```
Collective


## Input Parameters

- ***dm -*** The `DM`
- ***fv -*** The `PetscFV`



## Output Parameter

- ***dmGrad -*** The layout for gradient values
