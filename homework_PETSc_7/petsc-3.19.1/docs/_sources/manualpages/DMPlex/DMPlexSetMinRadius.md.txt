# DMPlexSetMinRadius
Sets the minimum distance from the cell centroid to a face 
## Synopsis
```
#include "petscdmplex.h"   
#include "petscfe.h"       
PetscErrorCode DMPlexSetMinRadius(DM dm, PetscReal minradius)
```
Logically Collective


## Input Parameters

- ***dm -*** the `DMPLEX`
- ***minradius -*** the minimum cell radius
