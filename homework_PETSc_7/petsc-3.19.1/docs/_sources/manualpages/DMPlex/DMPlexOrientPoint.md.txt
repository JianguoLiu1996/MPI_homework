# DMPlexOrientPoint
Act with the given orientation on the cone points of this mesh point, and update its use in the mesh. 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexOrientPoint(DM dm, PetscInt p, PetscInt o)
```
Not Collective


## Input Parameters

- ***dm -*** The `DM`
- ***p  -*** The mesh point
- ***o  -*** The orientation
