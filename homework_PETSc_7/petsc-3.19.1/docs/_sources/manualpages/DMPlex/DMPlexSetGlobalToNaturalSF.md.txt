# DMPlexSetGlobalToNaturalSF
Sets the `PetscSF` for mapping Global `Vec` to the Natural `Vec` 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexSetGlobalToNaturalSF(DM dm, PetscSF naturalSF)
```

## Input Parameters

- ***dm          -*** The `DM`
- ***naturalSF   -*** The `PetscSF`
