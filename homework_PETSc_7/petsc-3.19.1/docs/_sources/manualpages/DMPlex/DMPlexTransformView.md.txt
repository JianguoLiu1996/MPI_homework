# DMPlexTransformView
Views a `DMPlexTransform` 
## Synopsis
```
#include "petscdmplextransform.h" 
PetscErrorCode DMPlexTransformView(DMPlexTransform tr, PetscViewer v)
```
Collective


## Input Parameters

- ***tr -*** the `DMPlexTransform` object to view
- ***v  -*** the viewer
