# DMPlexSetChart
Set the interval for all mesh points [`pStart`, `pEnd`) 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexSetChart(DM dm, PetscInt pStart, PetscInt pEnd)
```
Not Collective


## Input Parameters

- ***mesh -*** The `DMPLEX`
- ***pStart -*** The first mesh point
- ***pEnd   -*** The upper bound for mesh points
