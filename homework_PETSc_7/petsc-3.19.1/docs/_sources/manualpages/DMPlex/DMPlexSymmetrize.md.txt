# DMPlexSymmetrize
Create support (out-edge) information from cone (in-edge) information 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexSymmetrize(DM dm)
```
Not Collective


## Input Parameter

- ***mesh -*** The `DMPLEX`





## Note
This should be called after all calls to `DMPlexSetCone()`
