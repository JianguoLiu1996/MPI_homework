# DMPlexGetPointDepth
Get the `depth` of a given point 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetPointDepth(DM dm, PetscInt point, PetscInt *depth)
```
Not Collective


## Input Parameters

- ***dm    -*** The `DMPLEX` object
- ***point -*** The point



## Output Parameter

- ***depth -*** The depth of the `point`
