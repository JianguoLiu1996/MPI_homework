# DMPlexGetDepthLabel
Get the `DMLabel` recording the depth of each point 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetDepthLabel(DM dm, DMLabel *depthLabel)
```
Not Collective


## Input Parameter

- ***dm    -*** The `DMPLEX` object



## Output Parameter

- ***depthLabel -*** The `DMLabel` recording point depth
