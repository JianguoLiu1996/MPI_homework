# DMPlexGetGlobalToNaturalSF
Gets the `PetscSF` for mapping Global `Vec` to the Natural `Vec`

```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetGlobalToNaturalSF(DM dm, PetscSF *naturalSF)
```

## Input Parameter

- ***dm          -*** The `DM`



## Output Parameter

- ***naturalSF   -*** The `PetscSF`
