# DMPlexMetricGetNormalizationOrder
Get the order p for L-p normalization 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexMetricGetNormalizationOrder(DM dm, PetscReal *p)
```

## Input parameters

- ***dm -*** The `DM`



## Output parameters

- ***p -*** The normalization order
