# TSForwardCostIntegral
Evaluate the cost integral in the forward run. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSForwardCostIntegral(TS ts)
```
Collective


## Input Parameter

- ***ts -*** time stepping context





## Note
This function cannot be called until `TSStep()` has been completed.
