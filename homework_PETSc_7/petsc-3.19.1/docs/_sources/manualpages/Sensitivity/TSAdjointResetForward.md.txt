# TSAdjointResetForward
Reset the tangent linear solver and destroy the tangent linear context 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSAdjointResetForward(TS ts)
```
Logically Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
