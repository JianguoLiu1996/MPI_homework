# TSAdjointReset
Resets a `TS` adjoint context and removes any allocated `Vec`s and `Mat`s. 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSAdjointReset(TS ts)
```
Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
