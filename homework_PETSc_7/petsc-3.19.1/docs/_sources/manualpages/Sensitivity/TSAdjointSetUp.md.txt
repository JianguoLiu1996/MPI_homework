# TSAdjointSetUp
Sets up the internal data structures for the later use of an adjoint solver 
## Synopsis
```
#include "petscts.h"  
PetscErrorCode TSAdjointSetUp(TS ts)
```
Collective


## Input Parameter

- ***ts -*** the `TS` context obtained from `TSCreate()`
